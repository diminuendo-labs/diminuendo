The function first replaces consecutive semicolons and commas to ensure the desired behavior for those edge cases. Then, it strips extra spaces from both sides of the input string, splits it by commas or semicolons (if no comma is found due to replaced semicolons), drops empty pieces at the start and end, and finally rejoins the remaining pieces with a single comma between each. If there are no pieces left after stripping extra spaces and dropping empty ones, an empty string is returned. This handles the edge case where only consecutive delimiters exist in the input.
