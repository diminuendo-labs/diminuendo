The function `median(nums)` takes a list of numbers as input and returns the middle value of the sorted list when the length is odd, or the average of the two middle values as a float when the length is even. In case of an empty list, it returns `None`. To handle edge cases, we check for an empty input list before proceeding with further calculations. The function first sorts the input list and then checks if the length is odd or even to determine whether to return a single value or the average of two values.
