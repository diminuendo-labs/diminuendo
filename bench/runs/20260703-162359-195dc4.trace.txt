I created a Python function `average(nums)` that calculates the arithmetic mean of a given list of numbers (`nums`). If the list is empty, it returns None instead of zero. I handled one edge case where the mean may be a whole number, but we still want it to return as a float. To do this, I used the `float()` function on the result of dividing the sum by the length of the list when all numbers are integers (to ensure the result is returned as a float). Otherwise, if there's no need to convert the mean to a float, I simply return the calculated average without conversion.
