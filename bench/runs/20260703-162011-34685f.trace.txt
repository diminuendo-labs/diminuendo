In this solution, I first convert the input list into a set to remove duplicate values and then calculate its length to get the total count of unique numbers. Next, I use a set comprehension to count the repeated numbers (those with a count greater than 1 in the original list). Finally, I subtract the count of repeated numbers from the total unique count to get the final result. This approach ensures that the function handles the edge case where all elements are unique or identical by returning 0 for an empty list and for lists containing only duplicates respectively. It also uses Python's built-in functions, set() and count(), making the code concise and readable.
