I created a function named `clamp` that takes three parameters: the `value`, and lower and upper bounds `lo` and `hi`. I first check if `lo` is greater than `hi`, in which case the range is degenerate, and return `None`. If the value is less than the lower bound or greater than the upper bound, I return the corresponding limit. Otherwise, the input value is within the specified bounds, so it's returned as-is. To make the code concise and readable, I used if-elif statements to handle each condition separately while ensuring clarity with proper indentation and variable names. An edge case considered in this implementation is when the input value falls outside the given range.
