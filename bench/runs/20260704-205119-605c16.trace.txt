The function first checks if the input list is empty and returns `None`. If it's not, it sorts the list in ascending order. It then finds the median index by dividing the length of the sorted list minus one by two and flooring the result. The median value is removed from both ends of the sorted list to ensure the trimmed mean calculation includes only a portion of the extreme values. Finally, it calculates and returns the mean of the remaining numbers in the list as a float. This function handles an edge case where the input list has an odd number of elements, ensuring that the median value is correctly identified and removed.
