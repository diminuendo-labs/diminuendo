I split the input string into parts using the underscore as a delimiter, which drops leading, trailing, and repeated underscores. I then capitalize each part (except the first one) to create the camelCase format. To handle the edge case of an empty string, I check if the length of the parts list is 1 and its only element is an empty string. If so, I return an empty string immediately. Additionally, I catch any ValueError that might be raised when capitalizing a part (which would indicate consecutive underscores) and raise it with a more informative message.
