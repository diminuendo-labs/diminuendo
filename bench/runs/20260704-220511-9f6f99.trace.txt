To write the `trimmed_mean()` function, I first handled empty and None inputs by returning None. Then, I sorted the input list to get the minimum and maximum numbers easily. To avoid removing the actual min or max values from the list, I skipped the first (index 1) and last but one (index len(nums)-2) elements while calculating the sum of the remaining numbers. Finally, I divided the total sum by the number of remaining elements to compute the trimmed mean as a float. The edge case I considered was when the list had less than three elements, in which case the function returns None.
