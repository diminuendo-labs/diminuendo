The function starts by sorting the input list and converting it to a set, which automatically removes duplicates, then back to a list for further processing. It then iterates through the sorted, unique numbers and checks if the current number is different from the next one in the list. If they are the same, it skips to the next iteration. If they are different, it adds the single number or the range (start-end) of consecutive runs to a result list. Finally, it joins all elements in the result list with commas and returns the string. An edge case handled is when the input list contains only one unique number. In that case, the function simply returns that number without a range.
