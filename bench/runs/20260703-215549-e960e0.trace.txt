I created the function `range_summary` that first checks if the input list is empty. If it is, it returns an empty string as specified in the edge case. Then, I sort and convert the list to a set to remove duplicates and sort them again. This ensures all numbers are distinct and ordered. Using a while loop, I iterate through the sorted list and identify consecutive runs by checking if each number is exactly one greater than the previous one. If a run is found, it's added as a range 'start-end' to the result list; otherwise, it's just a single number. Finally, I join all parts of the result list with commas and return the concatenated string.
