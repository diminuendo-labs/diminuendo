In this approach, we first create a dictionary called `freq` where each unique number from the input list is associated with its frequency count. The edge case handled here is that if there are duplicate numbers in the list, they will all be correctly counted as modes (numbers with maximum frequency). Then, we find the highest frequency using the built-in Python function `max()`. Next, we create a list of numbers that occur at the maximum frequency by filtering the dictionary items based on the maximum frequency count. Finally, we return the number of distinct modes by converting the list to a set and applying the built-in Python function `len()` on it.
