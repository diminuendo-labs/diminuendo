**
The function `range_summary` takes a list of integers and returns a string summarizing the distinct values in ascending order, collapsing consecutive runs into 'start-end'. The approach involves sorting the input list to easily identify consecutive numbers. I used two pointers (`start` and `end`) to track the current range being processed. When a non-consecutive number is encountered, the current range (or single number) is added to the result list in the appropriate format. Finally, any remaining range or single number is appended before joining all parts into a single string separated by commas. This approach ensures the function is both concise and readable while handling edge cases like an empty input list gracefully.
