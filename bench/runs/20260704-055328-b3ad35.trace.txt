I chose to use a list comprehension to concisely implement the function. The list comprehension iterates over each element `x` in the input list `lst`. If `x` is not a list (i.e., `not isinstance(x, list)`), it is included as-is in the output list. Otherwise, its first element (`x[0]`) is included. This implementation handles the edge case where the input list contains only one nested list with multiple elements; in this scenario, all but the first element of the inner list are ignored.
