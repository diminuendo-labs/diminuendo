I wrote a Python function named `chunk` that splits a given list into consecutive chunks of the specified size. The function takes two arguments, a list (`lst`) and an integer (`size`). It checks if the input size is a positive integer by using the `isinstance()` function and then iterates over the list in chunks of the provided size using a for loop with a generator expression. To handle the edge case where the list does not divide evenly, I used the range() function with three arguments to specify the starting index (0), increment (size) and the step value (same as size). If the size is zero or negative, it returns an empty list.
