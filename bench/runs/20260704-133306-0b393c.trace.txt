I started by checking if the input list is empty. If it is, I immediately return an empty list since there's nothing to chunk. Then, I use a list comprehension to create chunks of size `k` from the original list. The slicing operation `lst[i:i+k]` naturally handles creating these chunks in order. After creating the initial set of chunks, I check if the last chunk is smaller than `k`. If it is, I pad it with enough `None` values to make its length equal to `k`. This ensures that every chunk returned has exactly `k` elements.
