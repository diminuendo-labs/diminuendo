I started by checking if the input list is empty and returning an empty string if it is. This handles one edge case where the input might be an empty list. Then, I sorted the list to ensure that consecutive numbers are adjacent. I used two variables, `start` and `end`, to track the current range. As I iterated through the list, I checked if the current number was consecutive to the end of the current range. If it was, I updated the `end`. If not, I added the current range (or single number) to the result list and reset `start` and `end`. Finally, I handled any remaining range at the end of the loop and joined all ranges into a string separated by commas.
