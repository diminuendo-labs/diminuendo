The function calculates the weighted arithmetic mean by iterating over both lists in parallel using a generator expression, then summing up their products. To ensure that the function behaves correctly when given empty inputs or a zero total weight, it checks if the values and weights are empty and returns `None` in those cases.

An edge case handled by the function is when repeated weights occur in the input list; this can lead to incorrect results as each value should be multiplied by its respective unique weight only once. To handle this, the function raises a `ValueError` if the sum of the weights is not equal to the length of the values list.
