I wrote a function `count_token` that takes two arguments, `s` and `w`, representing the input string and target word respectively. The function splits the input string into individual words using whitespace as the delimiter and then uses a generator expression to count the number of words equal to the target word after stripping leading/trailing punctuation from each word and comparing case-insensitively. To handle edge cases, I added a docstring noting that an empty string returns 0. The function is concise and readable due to its simplicity and use of built-in string methods.
