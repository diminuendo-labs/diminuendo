The `range_summary` function first checks if the input list `nums` is empty and returns an empty string if true. It then sorts the unique elements of the list to ensure they are in ascending order. The function iterates through the sorted list, identifying consecutive numbers and forming ranges where applicable. When a non-consecutive number is encountered, it appends either the single number or the range to the summary list. Finally, it joins all parts of the summary with commas and returns the result.

The edge case handled here is an empty input list, which correctly returns an empty string.
