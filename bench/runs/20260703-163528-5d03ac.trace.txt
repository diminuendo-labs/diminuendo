I used the `collections.Counter` class to count the occurrences of each number in the input list. Then I found the mode (most frequent value) by accessing the last element of `most_common()`. The mode might not be unique, so I calculated the total sum of the mode(s), each counted once no matter how often it appears, by iterating through modes and adding that many times the difference between the length of the original list and the count of the current mode. In case the list is empty, it returns 0. To handle an edge case where there are multiple modes with the same frequency, I use the last mode (the one with the highest value) as the result.
