The function `median` takes a list of numbers as input and returns the median value according to the specified rules. If the list is empty, it returns `None`. To find the median, the function first sorts the list and then checks if the length of the list is odd or even. For an odd-length list, it returns the middle element; for an even-length list, it calculates the average of the two middle elements and returns it as a float. The approach ensures that the function handles both even and odd lengths correctly and provides a clear return value for an empty list.
