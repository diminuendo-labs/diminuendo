I chose to iterate through all possible pairs of numbers in the list and check if their sum equals the target. To avoid counting duplicate pairs, I used a set data structure. If a pair is found where both values are distinct (to account for edge cases such as [1, 2, 2] with target 3), I add it to the set using `frozenset` to ensure the added pair is immutable and thus easily hashable. Finally, the number of unique pairs is returned by the length of the set.
