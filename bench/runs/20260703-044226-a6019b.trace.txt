I used Python's `collections.Counter` to count the occurrences of each word. The function first checks if the input string is empty and returns an empty string if true. Otherwise, it converts the string to lowercase and splits it into words. It then counts the occurrences of each word and identifies the maximum count. A list comprehension filters out words that have this maximum count. Finally, the most common words (which could be more than one in case of a tie) are sorted alphabetically, and the first element is returned as the result. This approach ensures the function is both concise and readable while correctly handling ties by returning the word that comes first alphabetically.
