I implemented the `flatten_once` function by using a simple iterative approach. I initialized an empty list `result` to store the flattened elements, then iterated over each element in the input list. If an element is itself a list, I used the `extend` method to add its items to `result`. This effectively "flattens" one level of nesting without modifying deeper levels. The edge case for empty lists is handled by simply returning the original empty list as is.
