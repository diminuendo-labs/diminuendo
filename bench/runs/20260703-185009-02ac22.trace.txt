I chose to approach the problem by first splitting the input string into a list of words using the `split()` function. Then, I used the `map()` function with `len()` to find the maximum length of any word in the list. Next, I filtered the list of words with the `filter()` function to only include the longest words (those with the maximum length found earlier). To handle ties, I utilized the `default` argument in the filter function to set an empty string as the default return value when no longest words are found. Finally, I used the `max()` function to select the longest word out of the filtered list, ensuring that we prioritize the last occurrence based on the stated direction in the task's criteria.
