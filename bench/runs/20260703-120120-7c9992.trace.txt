I started by sorting the list to ensure that any consecutive numbers are adjacent. I then used a while loop to iterate through the sorted list, identifying segments of consecutive numbers. For each segment, if there is only one number, it is added as is; otherwise, the range "start-end" is added. The ranges and single numbers are collected in a list, which is then joined into a string with commas separating each element. This approach ensures that the function handles both individual numbers and sequences of consecutive numbers correctly.
