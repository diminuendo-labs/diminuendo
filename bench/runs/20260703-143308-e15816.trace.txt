**

I used the `Counter` class from Python's built-in `collections` module to count the frequency of each number. This made it easy to find the maximum count and the distinct numbers with that count. I then used a list comprehension to create a list of these modes, and finally converted this list to a set (which removes duplicates) before summing its elements. The edge case where an empty list is passed as input is handled by returning 0 immediately at the beginning of the function.
