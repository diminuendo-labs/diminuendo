I used Python's built-in `sorted()` function with a custom sorting key. The lambda function first sorts the words by length and then reverse-alphabetically by using the ASCII value of the last character (negated to sort in descending order). This ensures that words of equal length are ordered as required. Before performing any operations, I added a check to handle edge cases where `words` is empty or `k` is out of range, returning `None` in such scenarios.
