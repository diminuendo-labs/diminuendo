I approached this task by first splitting the input string `s` into a list of words using the `split()` method. Then, I initialized two variables - `min_len` to store the length of the shortest word and `result` to store the shortest word. I iterated over each word in the list, comparing its length with `min_len`. If a word is shorter than the current shortest, I updated `min_len` and `result`. On ties, since the task specifies returning the last such word, not the first, I chose to store the last encountered shortest word. In case there are no words in the input string, an empty string is returned according to the task requirements.
