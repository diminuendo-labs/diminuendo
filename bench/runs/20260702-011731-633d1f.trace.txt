The function `median` first checks if the input list is empty and returns `None` if true, handling the edge case where there are no elements to compute a median from. If the list is not empty, it sorts the list in ascending order. It then calculates the middle index of the sorted list. If the length of the list is even, it computes the average of the two middle numbers and returns that value as a float. If the length is odd, it returns the value at the middle index.
