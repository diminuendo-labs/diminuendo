I created a function that iterates through each character in the input string `s`. For every opening bracket, I increment the depth counter and check if it's greater than the current maximum depth. For every closing bracket, I decrease the depth counter only when the depth is greater than zero (meaning there are still open brackets). If a closing bracket appears before its opener or an opener is never closed, the function returns -1. Lastly, if the string is empty, it returns 0 as per the criteria. I chose this approach for its simplicity and readability, making it easy to understand how the function operates and handle edge cases such as brackets appearing out of order or missing closures.
