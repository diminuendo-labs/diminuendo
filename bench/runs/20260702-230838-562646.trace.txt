I implemented this function using a list comprehension to iterate over each element in the input list. The expression `isinstance(x, list)` is used to check if an element is a list; if it is not, it remains unchanged in the output list. For elements that are lists, I only take the first element (`x[0]`) using zero-based indexing, effectively "unwrapping" them at the first level of nesting. This function handles edge case where input `lst` contains nested lists with more than one element by keeping those inner lists intact, since they represent deeper nesting that should not be flattened.
