**  
The function `range_summary` takes a list of integers and returns a string with ranges of consecutive numbers. It first handles the edge case where the input list is empty, returning an empty string immediately. The list is then sorted and duplicates removed by converting it to a set and back to a list. We use two pointers (`start` and `end`) to identify consecutive sequences. As we iterate through the list, if the current number continues the sequence, we update `end`; otherwise, we add the range (or single number) to the result list and reset `start` and `end`. Finally, after the loop, we handle the last sequence and join all parts into a comma-separated string.
