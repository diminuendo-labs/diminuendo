The function `range_summary` first checks if the input list is empty and returns an empty string if so. It then sorts the unique elements of the list to handle both duplicates and sorting requirements. The main loop iterates through the sorted list, identifying consecutive ranges by checking if each element is one more than the previous. If a range is found, it appends the range in 'start-end' format; otherwise, it appends a single number. Finally, it joins all parts with commas to form the output string.

This approach ensures that the function handles all edge cases correctly and efficiently, including sorting, deduplication, and consecutive range identification.
