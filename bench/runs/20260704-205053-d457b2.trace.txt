The function first checks if the input list is empty, and returns `None` in that case. If not, it sorts the list and calculates the median index. The edge case considered here is an even-length list where the middle element might need to be ignored (by adjusting min_index and max_index). Afterward, it removes the minimum and maximum occurrences from the sorted list, sums up their values, divides by the number of remaining elements (n - 2), and returns the result as a float.
