The function `most_common_word(s)` first checks if the input string `s` is empty. If it is, an empty string is returned immediately as the edge case. The string is then converted to lowercase and split into individual words. A dictionary `word_count` is used to keep track of the frequency of each word. After populating the dictionary, the function finds the maximum count using `max(word_count.values())`. It then creates a list `most_common` containing all words that have this maximum count. Finally, it returns the lexicographically smallest word from `most_common`, or an empty string if `most_common` is empty. This approach ensures that the function is both correct and concise, adhering to the provided criteria.
