I started by splitting the input string `s` into a list of words using the `split()` method. This handles the edge case where the string is empty by returning an empty string if the resulting list is empty. To find the longest word, I used the built-in `max()` function with the `key=len` argument to find the word with the maximum length. Since `max()` returns the first item in case of a tie and the problem specifies to return the last such word, this approach naturally handles ties as desired.
