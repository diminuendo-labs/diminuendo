1. The function `least_frequent_word` takes a string `s` as input and processes it to find the word with the lowest frequency.
2. It first checks if the input string is empty, returning an empty string if true.
3. A dictionary `word_counts` is used to keep track of the count of each word in the string.
4. The string is split into words, converted to lowercase, and counts are updated accordingly.
5. The items in the dictionary are sorted first by frequency (ascending) and then alphabetically.
6. The function returns the first word from the sorted list, which will have the lowest count and, in case of a tie, the lexicographically smallest word.
7. An edge case is handled where the input string contains only one word. In this scenario, the function correctly identifies that word as having the least frequency since it's the only word available.
