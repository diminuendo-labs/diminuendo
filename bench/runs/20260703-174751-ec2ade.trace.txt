The function `count_pairs` takes a list of numbers and a target value as input. It iterates through each number in the list and calculates its complement (target - num). If the complement exists as a key in the dictionary `counts`, it adds the associated count to the result. If the current number is not already present in `counts`, it is added with an initial value of 0. Finally, the function returns the result. This approach uses O(n) space and O(n) time complexity. The edge case I named here is when there are multiple occurrences of equal values at different indices; they will still be counted as distinct pairs.
