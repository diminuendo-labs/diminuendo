The function `median(nums)` first sorts the input list `nums`. It checks if the length of the list is zero, returning `None` in that case. If the length is odd, it finds and returns the middle value as a float by dividing the index by 2 and rounding down. For even lengths, it calculates and returns the average of the two middle values by adding them and dividing by 2. An edge case handled here is when the list is empty.
