I approached this task by first replacing consecutive commas with a single comma to make the splitting easier. Then I used Python's built-in `split()` function to split the string on commas, and dropped empty pieces at the start and end using list comprehension. Finally, I rejoined the remaining pieces with a single comma between each using another call to `join()`. An edge case that this function handles is when there are multiple consecutive commas in a row within the input string.
