I first checked for an empty list and returned None if it was the case. Then, I sorted the list in ascending order. After that, I calculated the indices of the minimum and maximum values, excluding one occurrence each by subtracting 1 from the first index and using the length minus the new first index minus 2 for the second index. Finally, I summed up the elements between those two excluded indices and divided the result by the number of elements left in the original list, effectively calculating the trimmed mean.
The edge case named is when the input list is empty, in which case the function returns None.
