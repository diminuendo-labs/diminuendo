I created a function `snake_to_camel(s)` that first replaces all underscores with nothing and converts the whole string to lowercase. Then, I split the resulting string into words by underscores. The loop iterates through each word except for the first one, capitalizing it and appending to `result`. This approach ensures that leading, trailing, and repeated underscores are dropped while maintaining camelCase formatting. The edge case named is when there's more than one consecutive underscore in the input string.
