The function `range_summary` first checks if the input list is empty and returns an empty string if true. It then sorts the unique numbers in ascending order. Using a while loop, it iterates through the sorted list to identify consecutive runs of numbers. For each run, it either appends the single number or the range as a string to the result list. Finally, it joins all elements of the result list with commas and returns the formatted string. The edge case handled is an empty input list, which results in an immediate return of an empty string.
