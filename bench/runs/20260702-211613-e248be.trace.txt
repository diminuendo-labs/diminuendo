The function `count_distinct_pairs` takes a list of integers `nums` and an integer `target`. It first handles the edge case where the input list is empty by returning 0 immediately. The solution uses two sets: `seen` to keep track of numbers encountered so far, and `pairs` to store unique pairs that sum up to the target. As it iterates through the list, for each number, it calculates its complement with respect to the target. If the complement has been seen before and is not already in the `pairs` set (to ensure uniqueness), it adds a sorted tuple of the pair to `pairs`. Finally, it returns the count of unique pairs found.
