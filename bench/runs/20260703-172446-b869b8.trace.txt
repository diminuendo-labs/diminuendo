I wrote a Python function `median(nums)` that first sorts the given list of numbers. The middle index of the sorted list is found by dividing the length by 2. If the number of elements in the sorted list is odd, the function returns the middle value as a float. Otherwise, it calculates and returns the average of the two middle values as a float. An empty list results in None being returned to handle the edge case. The code is concise, readable, and easy to understand with clear variable names.
