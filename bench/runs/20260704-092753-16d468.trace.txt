I used a Stack data structure to keep track of opening parentheses encountered in the string. Whenever a closing parenthesis is found, it checks if there is an opening parenthesis on top of the stack or if the stack is empty. If either condition is true, it returns False. The function returns True only when the stack is empty at the end, meaning all parentheses have been matched properly. I handled an edge case where an unexpected closing parenthesis appears before its corresponding opening parenthesis.
