I implemented the function named `clamp` that takes three parameters - `value`, `lo`, and `hi`. The first check ensures that the range is not degenerate by checking if `lo` is greater than `hi`. If it is, then the function returns `None`. The second and third checks handle the edge case where the input value is less than `lo` or greater than `hi` respectively, by returning the appropriate bound. Otherwise, the original value is returned if it falls within the range. I chose to use simple conditional statements for readability and conciseness.
