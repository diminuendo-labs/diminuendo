To create the `median` function, I first defined it to accept a list of numbers and check if the list is empty. If so, the function returns None. Then, I sorted the list using the built-in `sorted()` function in Python. After that, I determined whether the length of the sorted list was odd or even by checking the remainder when divided by 2.

If the length was odd, I returned the middle value by finding the index of the middle element with the `n // 2` floor division operation. If the length was even, I calculated the average of the two middle values using arithmetic mean (addition and division). This approach ensures that the function works correctly for both odd and even lengths of input lists, handles edge cases, and provides a concise and readable solution.
