I implemented the function using a stack data structure to keep track of opening parentheses. When an opening parenthesis is encountered, it's pushed onto the stack; when a closing parenthesis is encountered, the stack is checked for emptiness and if it's not empty, the top element (the corresponding opening parenthesis) is popped. After iterating over all characters in the string, the function returns whether the stack is empty or not to indicate balance. I named one edge case explicitly: an empty string is balanced.
