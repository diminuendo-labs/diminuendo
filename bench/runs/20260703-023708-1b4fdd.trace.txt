I chose to approach this task using a stack data structure. The function iterates through each character in the input string, pushing opening parentheses onto the stack and popping them off when corresponding closing parentheses are encountered. If a closing parenthesis appears before its matching opening partner (i.e., the stack is empty), the function immediately returns False. Otherwise, after iterating through all characters, if there are still items on the stack (i.e., unmatched opening parentheses), the function returns False; otherwise, it returns True. This approach efficiently handles edge cases such as an empty string, which is correctly considered balanced.
