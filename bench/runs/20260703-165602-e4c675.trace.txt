First, I created an empty dictionary `counter` to store the count of each number in the list. Then, I iterated through the list and updated the count for each number, also keeping track of the maximum frequency encountered. After that, I iterated again through the counter to find all numbers with the highest frequency and increment the modes variable for each one found. Finally, I return the count of modes. The edge case addressed is when the input list is empty, in which case no modes are present so 0 is returned.
