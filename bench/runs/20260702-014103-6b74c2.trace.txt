** I created a function `flatten_once` that iterates through each element of the input list. If an element is a list itself, it extends the resulting flattened list with its items using `extend()`. Otherwise, it simply appends the item to the result. This approach ensures that only one level of nesting is flattened, while deeper levels remain nested. The function handles the edge case where the input list contains non-list elements by treating them as single items and appending them directly to the result list.
