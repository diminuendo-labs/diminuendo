** The function `average` calculates the arithmetic mean of a list of numbers. It first checks if the input list `nums` is empty. If it is, the function returns `None`. Otherwise, it computes the sum of the list using the built-in `sum()` function and divides it by the length of the list using the built-in `len()` function to get the mean. The result is returned as a float, which will automatically be a whole number if there are no decimal places.
