**: The function `count_distinct_pairs` iterates through the list of numbers. For each number, it calculates its complement with respect to the target sum. If the complement has been seen before, it adds a sorted tuple of the pair to a set to ensure uniqueness and order. This avoids counting duplicate pairs regardless of their frequency in the input list. Finally, the function returns the size of the set containing all unique pairs. The edge case for an empty list is handled by returning 0 immediately.
