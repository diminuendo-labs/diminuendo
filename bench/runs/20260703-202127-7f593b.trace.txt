The function takes a list of numbers as input and returns the second largest distinct value, or `None` when there are fewer than two distinct values. It initializes `max1`, `max2`, and an empty set `seen`. The for loop iterates through each number in the given list and checks if it is greater than `max1`, `max2`, or already seen. If a number is greater than `max1` but less than or equal to `max2`, it updates `max2` with that number. If a number equals `max1`, it returns `None` (edge case), as there can only be one largest value in the list. If a number already exists in the set `seen`, it means that the list contains duplicate values, and again, we return `None`. If no such conditions are met, the function adds the number to `seen` and proceeds to the next iteration until all numbers have been processed. Finally, the function returns `max2` which is either the second largest distinct value or `None` if there were not enough distinct values in the list.
