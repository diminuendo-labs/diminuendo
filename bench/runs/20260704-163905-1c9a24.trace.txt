The function `chunk_pad` takes a list `lst` and an integer `k`, where `k` is the size of each chunk. It first checks if `k` is greater than zero to handle the edge case where the chunk size might be non-positive, which would not make sense for chunking. The function then uses a list comprehension to create chunks of size `k` from the list. For each chunk, it pads with `None` elements on the right until the chunk reaches size `k`. This ensures that every chunk in the output has exactly `k` elements. If the input list is empty, the function simply returns an empty list as required.
