I chose a simple and concise approach using a list comprehension. This allows me to iterate over each element `x` in the input list `lst`, checking if it's a list or not with `isinstance(x, list)`. If it is a list, I extract its first element `x[0]`; otherwise, I keep `x` as is. This handles edge cases where lists are empty or have only one element, effectively preventing deeper nesting from being flattened.
