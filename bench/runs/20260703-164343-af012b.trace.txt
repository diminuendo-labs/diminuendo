First, I split the input string into words using the `split()` method. Then, I loop through each word, checking if it is entirely uppercase by using the `isupper()` method. If not, I capitalize the first letter of the word and make the rest lowercase using the `capitalize()` method. Finally, I join all the words back into a single string using the `join()` method. This approach ensures that words already entirely uppercase stay exactly as they are while handling other cases according to the criteria.
