I started by splitting the input string on the underscore character and stripping any leading or trailing underscores. To handle the edge case of repeated underscores, I checked if the length of the split list was equal to the length of a set created from that list (which would have removed duplicates), raising a ValueError if so. Finally, I joined the resulting parts with their first letter capitalized using a list comprehension and the `join` method. This solution is concise while still being readable due to the clear function name, docstring, and variable names.
