In this solution, we first define an exception list to preserve the given form for 'URL' and 'ID'. If the input string contains any of these exceptions as the first token, we simply return it as is. Otherwise, we split the input string by underscores and convert the first word to lowercase while capitalizing the remaining words. Finally, we join all the transformed tokens together without a separator to obtain the camel-cased identifier. An edge case handled here is when an exception token appears somewhere other than the beginning of the string.
