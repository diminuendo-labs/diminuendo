- The function `median` first checks if the input list `nums` is empty and returns `None` if it is.
- If the list is not empty, it sorts the list using Python's built-in `sorted()` function.
- Depending on whether the length of the sorted list (`n`) is odd or even, the function calculates and returns the median accordingly:
  - For an odd number of elements, it returns the middle element.
  - For an even number of elements, it calculates the average of the two middle elements.
