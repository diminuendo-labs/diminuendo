To implement the `range_summary` function, I first check if the input list `nums` is empty. If it is, I return an empty string as specified in the edge case. Then, I convert the list to a set and sort it to ensure that the numbers are in ascending order and duplicates are removed. I use a while loop to iterate through the sorted numbers, identifying consecutive ranges by checking if the next number is exactly one more than the current number. If a range is identified (more than one number), it is added to the result as 'start-end'. If not, the single number is added as is. Finally, I join all parts of the result with commas and return the final string. This approach ensures that the function is both correct and concise, while also handling the edge case of an empty input list.
