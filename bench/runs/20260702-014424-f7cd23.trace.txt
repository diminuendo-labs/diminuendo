I chose to iterate through the input list and check the type of each element. If an element is a list, I extended the result list with its items; otherwise, I appended the item as is. This approach ensures that only one level of nesting is flattened while preserving deeper levels of nested lists. The edge case handled is when encountering empty lists or non-list elements, which are left unchanged to avoid any unintended behavior during flattening.
