I approached this task by first replacing any double commas with a single comma to simplify the splitting process. Then, I used list comprehension to select only the non-empty pieces from the split string and finally re-joined them using a single comma between each. To handle an edge case where there might be extra leading or trailing empty strings due to consecutive commas, I made sure to exclude those while joining the final result.
