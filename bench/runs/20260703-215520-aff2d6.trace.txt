I started by sorting the list to ensure that consecutive numbers are adjacent. I then iterated through the sorted list, keeping track of the start of a sequence. When I encountered a number that was not consecutive to the next one, I checked if it was a range (more than one number). If it was a range, I formatted and added it to the result list; otherwise, I added the single number. Finally, I joined all parts of the result into a string separated by commas. An empty input list is handled by immediately returning an empty string.
