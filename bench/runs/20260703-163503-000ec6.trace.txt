The function uses the `Counter` class from Python's collections module to count the occurrences of each number in the list. The `most_common()` method returns a list containing the most frequent items and their counts, sorted in descending order by count. The function iterates through this list, adding up the values that appear only once (when there is no multiple mode) or counting each multiple-occurring mode only once. Lastly, it ensures that an empty list returns 0 and handles the edge case where multiple modes exist by subtracting one from the count of those numbers.
