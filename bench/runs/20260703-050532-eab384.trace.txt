I started by sorting the list and removing duplicates using `sorted(set(nums))` to ensure that each distinct value is considered only once. I then iterated through the sorted numbers, maintaining a start and end pointer for consecutive sequences. When a non-consecutive number is encountered, the current range (if any) is added to the result list, and the pointers are updated. Finally, the remaining range is appended, and all ranges are joined into a comma-separated string before being returned. The function handles the edge case of an empty input list by returning an empty string.
