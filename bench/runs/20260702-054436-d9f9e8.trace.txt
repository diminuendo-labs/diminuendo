The function `snake_to_camel` takes a string `s` as input and converts it from snake_case to camelCase. It first removes any leading or trailing underscores using `strip('_')`. The string is then split into words based on underscores using `split('_')`, and an empty string filter is applied with `if word` to ensure no empty strings are processed further. Each word is capitalized using `capitalize()` method, and they are joined together without any separators to form the camelCase string. This approach handles leading, trailing, and repeated underscores as specified in the criteria, and an edge case where the input string is entirely made of underscores (e.g., '__a__b_') will return an empty string as intended.
