I implemented the `flatten_once` function by iterating over each element in the input list. If an element is a list itself, I used a list comprehension to separate non-nested elements and recursively call `flatten_once` on its nested list part. This approach ensures that only one level of nesting is flattened, while preserving deeper nesting as it is. The edge case handled by this function is when the input list contains other types of elements besides lists (e.g., integers, strings), which will be kept unchanged in the output list.
