In this function, we first define a list of exceptions to preserve the given form for tokens 'URL' and 'ID'. We then split the input string by underscores and iterate through each token. If the token is not in the exception list and it's not an empty string, we convert it to title case before appending it to our camelCase string with no separator. Finally, we return the camel-cased string with the first letter capitalized. The edge case handled here is when there are multiple consecutive underscores in the input string. Since the function replaces all underscores with no separators, it will handle that situation as expected (e.g., 'multi_part_identifier' becomes 'multiPartIdentifier').
