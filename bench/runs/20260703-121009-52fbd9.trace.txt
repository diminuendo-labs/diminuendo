The function `range_summary` first checks if the input list is empty and returns an empty string if true. It then sorts the unique elements of the list to ensure they are in ascending order. The function iterates through the sorted list, grouping consecutive numbers into ranges. For each group, it appends either a single number or a range (start-end) to the result list. Finally, it joins all parts of the result list with commas and returns the string. This approach ensures that the output is correctly formatted as specified in the criteria.
