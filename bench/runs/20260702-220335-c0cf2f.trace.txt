I approached this task by using Python's regular expressions module `re` to split the input string `s` into individual words while ignoring punctuation. This ensures that substrings inside longer words do not count towards the total. Then, I used a generator expression within the `sum` function to count how many of these processed words match the target word `w`, after stripping leading and trailing punctuation and converting both to lower case for case-insensitive comparison. The edge case where an empty string is input was handled by returning 0 immediately if the stripped string is empty.
