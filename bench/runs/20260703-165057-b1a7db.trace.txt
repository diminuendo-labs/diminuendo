The function `least_frequent_word(s)` first checks if the input string `s` is empty, and returns an empty string in that case. Then, it splits the string into individual words (assuming whitespace as a separator), counts their occurrences using a dictionary, and finds the minimum count among all words.

Next, it collects all words having the least frequency and sorts them alphabetically. If there are multiple words with the same least frequency, the first one in the sorted list is returned. This handles the edge case where multiple words have the same lowest count.
