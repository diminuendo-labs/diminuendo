In this solution, I first handle the edge case where the input list is empty or contains fewer than 3 elements by returning None. Then, I sort the list and check if both minimum and maximum occurrences appear only once. If they do, I remove them from the list using `pop()`. After that, I filter out any `None` values from the list using `filter(None, nums)`. Lastly, I calculate the trimmed mean as the sum of remaining non-zero elements divided by the number of remaining elements minus 2 (since we removed one minimum and one maximum).
