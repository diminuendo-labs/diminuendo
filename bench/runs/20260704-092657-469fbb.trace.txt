The function `balanced(s)` takes a string `s` as input and uses a stack data structure to check the balance of parentheses. Each time an opening parenthesis '(' is encountered, it is added to the stack. When a closing parenthesis ')' appears, if the stack is empty or the top element in the stack is not an opening parenthesis, the function returns `False`. The function returns `True` only when there are no unmatched parentheses and the stack is empty at the end. An edge case handled here is when the input string contains extra non-parenthesis characters.
