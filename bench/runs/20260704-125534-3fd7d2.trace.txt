In this solution, we first create an exception list containing the identifiers that should remain unchanged ('URL' and 'ID'). The input string is then split by underscores to get a list of tokens. We iterate through each token and check if it is in the exception list. If not, we convert the first character to lowercase and the rest to uppercase, and append it to the result list. After that, we join all the resulting tokens with no separator. To handle an edge case, we also added checks for when the input string is empty or contains only whitespaces, in which cases we return an empty string instead of raising any errors. This makes our function robust and easy to use with different types of inputs.
