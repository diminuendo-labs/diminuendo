I implemented the function by first splitting the input string 's' on underscores ("_") into a list of words. Then, I initialized a result variable to the first word in the list (to handle the case when the input is empty). After that, I iterated over the rest of the words (excluding the initial one) and concatenated each word with its first letter capitalized to the result. Finally, I returned the result or an empty string if it was empty initially. The edge case handled in this solution is when the input string is an empty string.
