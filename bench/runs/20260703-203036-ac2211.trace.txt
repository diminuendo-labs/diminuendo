The function `second_largest` takes a list of numbers as input and returns the second largest distinct value, or `None` when there are fewer than two distinct values. The approach used is to first sort the list in descending order using Python's built-in `sort()` method. If the length of the sorted list is less than 2, it means that there are less than two unique numbers, and thus we return `None`. Otherwise, we return the second element (index 1) from the sorted list which represents the second largest number. The edge case I've considered here is when the list contains duplicates of the first or second largest numbers; in this situation, the function will still correctly identify the second largest unique value.
