The function `range_summary` first checks if the input list is empty and returns an empty string if it is. Otherwise, it sorts the list to ensure that consecutive numbers are adjacent. It then iterates through the sorted list, tracking the start and end of a potential range. When it encounters a number that is not consecutive (i.e., not equal to `end + 1`), it appends either a single number or a range (`start-end`) to the result list, depending on whether the start and end are different. After processing all numbers, it handles the last range in the same way as before. Finally, it joins all elements in the result list with commas and returns the string. This approach ensures that the function is concise, readable, and correctly handles edge cases such as an empty input list or a list with a single number.
