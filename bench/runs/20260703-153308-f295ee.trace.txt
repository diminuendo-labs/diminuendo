The function `range_summary` first sorts the unique elements of the input list `nums`. It then iterates through the sorted list, grouping consecutive numbers into ranges. When a non-consecutive number is encountered, it appends either a single number or a range to the result list accordingly. The final step joins all parts of the result with commas and returns the string. This approach ensures that the function correctly handles consecutive sequences by collapsing them into 'start-end' format while keeping single numbers as they are.
