I used a stack data structure to keep track of the opening parentheses that have not yet been closed. As I iterate through the string, when I encounter an opening parenthesis, I push it onto the stack. When I encounter a closing parenthesis, I check if the stack is empty (which would indicate that the closing parenthesis appears before its partner) and return False if so. If the stack is not empty, I pop the corresponding opening parenthesis from the stack. Finally, after iterating through the entire string, I return True if the stack is empty (meaning all parentheses were balanced) or False otherwise.
