I created the `is_anagram` function to determine if two input strings are anagrams. Anagrams are words or phrases that contain the same characters in any order. To ensure the comparison is case-insensitive and not affected by spaces, I first removed all spaces from each string using `replace(" ", "")` and then converted them to lower case with `.lower()`. Finally, I sorted the characters of both normalized strings and compared them for equality. This method ensures that the function correctly identifies anagrams while ignoring differences in case and spacing. The edge case this function implicitly handles is when either or both input strings are empty; in such cases, the function will return `True` because sorting an empty string results in another empty string, which are considered equal.
