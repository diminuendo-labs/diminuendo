** The function `median` first checks if the input list `nums` is empty and returns `None` if it is, handling an edge case. If not, it sorts the list using Python's built-in `.sort()` method for efficiency. Depending on whether the length of the list is odd or even, it calculates and returns the median accordingly: the middle element for odd lengths, or the average of the two middle elements for even lengths. The division by `2.0` ensures that the result is a float when necessary.
