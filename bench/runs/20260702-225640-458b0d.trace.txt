I used the `Counter` class from the `collections` module to count the occurrences of each word in the input string. This allows me to efficiently find the most common word(s). To break ties alphabetically, I use the fact that `most_common(1)` returns a list of tuples containing the most common elements and their counts; since there's only one tuple, I can directly access its first element (`max_word`) and ignore the second element (`_`), which is the count. The edge case for an empty string is handled by checking if the input string is stripped of whitespace before processing it.
