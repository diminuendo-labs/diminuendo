I chose to use a stack data structure to keep track of the opening parentheses encountered so far. As I iterate over the input string, whenever I encounter an opening parenthesis, I push it onto the stack. When I encounter a closing parenthesis, I check if the stack is empty (in which case the string is not balanced) or pop the top element from the stack. Finally, after iterating over the entire string, I return True if the stack is empty (i.e., all parentheses were properly closed) and False otherwise. This approach handles the edge case of an empty string as expected, returning True since there are no unbalanced parentheses.
