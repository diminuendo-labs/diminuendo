I created a dictionary called `frequency` to store the counts of each number in the list. The edge case I considered is when the input list is empty, in which case the function returns 0. To find the mode and its count, I first identified the maximum frequency value, then iterated through the dictionary to count how many numbers have that frequency. Each number that shares the highest frequency will be counted only once by incrementing the `mode_count` variable each time we encounter a number with the maximum frequency.
