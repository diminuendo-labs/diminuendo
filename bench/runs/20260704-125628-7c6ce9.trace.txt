I approached this task by first defining a set of exceptions to keep the given form of the tokens 'URL' and 'ID'. I then split the underscore-delimited identifier into individual tokens. For each token, I checked if it was in the exception list; if not, I capitalized the first character of the token and added it to the camelCase list (except for the first token which I made lowercase). Finally, I joined all the tokens in the camel_case list into a single string. To handle an edge case where an empty token might appear due to string trimming operations, I added a check to ensure that each token is non-empty before processing it.
