I wrote the `trimmed_mean` function that calculates the mean of a list after removing exactly one occurrence of the minimum and maximum values. I used Python's built-in sort() method to sort the list in ascending order and then chose the appropriate indices based on whether the list has an even or odd number of elements. If the list has an even number of elements, I removed the second smallest or largest value by setting it to None, sorted the list again, and then calculated the mean by summing up the remaining values and dividing by the total count minus 2. The function returns None if the input list is empty.
