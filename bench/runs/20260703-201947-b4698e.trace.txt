I created a Python function called `second_largest(nums)`. The function first checks if there are fewer than two distinct values in the input list, and returns `None` when that's the case. If there are more than one unique value, it sorts the list in descending order (reverse=True). Finally, it returns the second largest element or None if the list only has one element. The purpose of sorting the list is to ensure that the largest and second-largest elements end up at the last two positions, making them easily accessible.
