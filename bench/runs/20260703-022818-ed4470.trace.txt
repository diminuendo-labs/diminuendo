I chose to use a stack data structure to keep track of the opening parentheses encountered so far. For each character in the input string, if it's an opening parenthesis, I push it onto the stack; if it's a closing parenthesis, I check if the stack is empty (in which case there's no matching opening parenthesis) or pop the top element from the stack. Finally, after iterating over all characters, I return True if the stack is empty (meaning every opening parenthesis was matched by a closing one) and False otherwise. This approach ensures that parentheses are balanced without having to keep track of their indices in the string.
