I chose to use Python's built-in `split()` function to split the input string into a list of words. This function splits on whitespace by default, which meets the requirements. To handle ties and ensure that the last such word is returned, I used the `max()` function with the `key` argument set to `len`, specifying no initial value (i.e., `default=''`). This allows `max()` to return an empty string when the input is a single space or whitespace only.
