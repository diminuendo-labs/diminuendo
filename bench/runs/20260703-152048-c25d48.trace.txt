The function `range_summary` first handles the edge case where the input list is empty, returning an empty string. It then sorts the unique elements of the list to ensure proper order and preparation for the range collapse logic. The main loop iterates through the sorted numbers, identifying ranges by checking if the current number is not consecutive to the previous one. Depending on whether a range was identified or just a single number, it appends the appropriate string to the `ranges` list. Finally, it joins all the strings in the `ranges` list with commas and returns the result.
