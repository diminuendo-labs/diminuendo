The function `median` first checks if the input list is empty and returns `None` as required for the edge case. It then sorts the list to easily find the middle value(s). If the length of the sorted list is even, it calculates the average of the two middle numbers by summing them and dividing by 2.0 to ensure a float result. For an odd length, it simply returns the middle number as a float. This approach ensures the function works correctly for all edge cases, including handling empty lists.
