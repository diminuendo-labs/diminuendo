The function `trimmed_mean` first checks if the list has two or fewer elements, in which case it returns `None`. Otherwise, it removes one occurrence of both the minimum and maximum values from the list. Finally, it calculates and returns the mean of the remaining numbers as a float.

The edge case this function handles is when the input list is empty, a one-element list, or a two-element list, all of which result in `None` being returned.
