I approached this problem by first checking for the edge case of an empty list, which returns an empty string. Then I sorted the input list to ensure that consecutive numbers are adjacent. I used a single loop to iterate through the sorted list and build ranges by keeping track of the current start and end values. When a number is not one more than the previous number, it means we've hit a new range or the last number in a range, so I append the current range (if it's not just one number) or update the current range to be just that single number. Finally, I join all ranges with commas and return the result.
