The function `longest_word(s)` splits the input string into words using `s.split()`. It then calculates the maximum length of these words. Next, it filters out the words that have this maximum length into a list called `longest_words`. Finally, it returns the last word in this list, which is the longest word according to the tie-breaking rule specified (the last such word on ties). This approach ensures that the function adheres to the requirement and handles edge cases appropriately.
