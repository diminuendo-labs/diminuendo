I chose to use Python's built-in `split()` method to split the input string into a list of whitespace-separated words. To handle ties, I used a generator expression within a call to `next()`, which returns the first item from an iterable for which a given condition is true; in this case, the condition is that the word's length equals the maximum length found so far. This ensures that if multiple words have the same maximum length, the last such word (i.e., the one closest to the end of the list) is returned. I also explicitly checked for an empty string as input and returned an empty string in this case.
