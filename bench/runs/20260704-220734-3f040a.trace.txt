I used Python's built-in `sort()` method to sort the input list in ascending order. Then, I checked if the length of the list is 1 or 0, returning the only number in such cases. For the remaining cases, I calculated the mean by either summing all but the first and last numbers (if the list length is even) or summing all but the middle number along with the second-last number (if the list length is odd). This ensures that one occurrence of the minimum and maximum values are excluded from the calculation. The edge case handled here is when the input list has only one number.
