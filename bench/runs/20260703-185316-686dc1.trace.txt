The function first splits the input string into a list of words. Then it finds the maximum length of any word using a generator expression. Finally, it returns the longest word by filtering the list with the lambda function that checks if the length of a word matches the maximum length found earlier. In case of ties, it returns the last such word due to Python's default sort order which is stable and maintains the relative ordering of equal elements. The edge case I handled here is when there are no words in the input string (an empty string), in which case the function returns an empty string as well.
