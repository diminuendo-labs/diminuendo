I started by checking if the input string `s` is empty, returning an empty string immediately if it is. Then I split the string into a list of words using the `split()` method. To find the maximum word length, I used a generator expression inside the `max()` function. Next, I created a list comprehension that iterates over the words in reverse order (to ensure the last longest word is selected in case of ties) and filters out the words whose lengths match the maximum length found earlier. Finally, I returned the first element of this filtered list if it exists, otherwise an empty string.
