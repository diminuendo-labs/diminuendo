The function first sorts the input list `nums`. It checks if the list is empty and returns `None` in that case. Then, it calculates the index of the middle element(s) based on whether the length of the list is odd or even. For an odd-length list, it simply returns the middle value. For an even-length list, it averages the two middle values by adding them and dividing by 2 (as a float). The edge case handled here is when the input list is empty.
