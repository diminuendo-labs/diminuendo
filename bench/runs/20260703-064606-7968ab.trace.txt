I approached this task by using Python's built-in `split()` function to split the input string into a list of words. I then used the `max()` function with the `key` argument set to `len` to find the longest word in the list. To handle ties, I specified that if there are multiple longest words, only return the last one. Finally, I included an edge case note in the docstring about what happens when the input string is empty.
