To solve this problem, I followed these steps:

1. **Check for Edge Case**: First, I check if the input list `nums` is empty. If it is, I immediately return an empty string.

2. **Sort and Remove Duplicates**: I convert the list to a set to remove duplicates and then sort it in ascending order. This ensures that any consecutive numbers are adjacent.

3. **Iterate Through the List**: Using a while loop, I iterate through the sorted list. For each number, I check if there is a consecutive sequence starting from it by comparing the current number with the next one. If a sequence is found, I append 'start-end' to the result list; otherwise, I append the single number.

4. **Join and Return**: Finally, I join all elements in the result list with commas and return the string.

This approach ensures that consecutive numbers are collapsed into ranges, non-consecutive numbers remain as single numbers, and the function handles empty input gracefully.
