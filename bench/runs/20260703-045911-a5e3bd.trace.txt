The function `range_summary` first checks if the input list `nums` is empty. If it is, it returns an empty string immediately to handle this edge case. The list is then sorted and duplicates are removed by converting it to a set and back to a list. We use a while loop to iterate through the sorted numbers. For each number, we determine if there is a consecutive range by checking the next element in the list. If a range is found, we append the 'start-end' format to the summary list; otherwise, we add the single number. Finally, we join all parts of the summary with commas and return the result. This approach ensures the code is both concise and readable while correctly handling consecutive ranges and unique values.
