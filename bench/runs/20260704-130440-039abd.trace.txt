First, I defined an exception list containing the strings 'URL' and 'ID'. Then, I initialized `camel_case` by lowercasing the input string and capitalizing its first letter. After that, I split the remaining tokens using underscores and iterated through them. If a token was not in the exception list and it was not empty, I appended the capitalized version of the token to `camel_case`. The edge case considered is when the input string starts with an exception token, which will keep its exact form even when it's the first token.
