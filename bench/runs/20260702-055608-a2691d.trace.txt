The `median` function first checks if the input list `nums` is empty and returns `None` in that case. If not, it sorts the list using Python's built-in `sorted()` function. It then calculates the middle index. If the length of the list is even, it returns the average of the two middle numbers by summing them and dividing by 2.0 to ensure a float result. If the length is odd, it simply returns the value at the middle index. This approach ensures correctness, conciseness, and readability while handling the edge case of an empty list.
