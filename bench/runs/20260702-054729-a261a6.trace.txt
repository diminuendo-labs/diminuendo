I approached this task by using the `sum` function with a generator expression. The generator iterates over each element in the input list; if an element is a list, it yields its items directly; otherwise, it yields the item itself unchanged. This way, exactly one level of nesting is flattened while deeper levels remain intact. I chose this method because it's concise and leverages Python's built-in functions for efficient processing. The edge case of empty lists or non-list elements is naturally handled by the generator expression since they will simply pass through without modification.
