This solution uses the built-in Python functions `set()` and `len()` to remove duplicates and count elements of the set, which gives us the total number of unique values in the list. To handle the requirement that only values appearing exactly once should be counted as unique, we calculate the length of a new set containing each number from the original list, with the frequency of each number being 1 (i.e., {num for num in nums if nums.count(num) > 1} will only contain duplicates). Finally, we subtract the count of this set from the initial set length to get the desired result. The edge case considered here is handling lists with no elements where a return value of 0 is expected.
