To implement the `collapse_delims` function, I used Python's regular expression module (`re`) to split the input string `s` by one or more consecutive commas or semicolons. The `strip()` method is applied to remove any leading or trailing whitespace before splitting. The resulting list comprehension filters out any empty strings that may result from consecutive delimiters. Finally, the pieces are joined back together with a single comma between each using the `join()` method. This approach ensures that the function handles edge cases such as multiple consecutive delimiters and leading/trailing spaces correctly.
