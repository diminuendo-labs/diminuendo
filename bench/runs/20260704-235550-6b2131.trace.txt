I implemented the function `kth_ordered` to return the k-th word from a list of words, ordered by increasing length and reverse-alphabetically for words of equal length. The solution uses Python's built-in `sorted` function with a custom key. The key sorts primarily by negative length (to sort by increasing length in ascending order) and secondarily by reverse alphabetization (using `x[::-1]`). This ensures that words of the same length are sorted in reverse-alphabetical order. The function also handles an edge case where the input list is empty, returning `None` in such cases.
