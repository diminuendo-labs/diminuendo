I first defined the set of delimiters as a string containing commas, semicolons, and spaces. Then I used Python's built-in `split()` method to split the input string by these delimiters and removed leading and trailing whitespaces.
Next, I iterated over the list of pieces until consecutive empty pieces at the start or end were fully removed from each piece. The key here was to avoid an infinite loop in case there are multiple consecutive empty pieces. Once all empty pieces were removed, I joined the remaining non-empty pieces back together using a single comma as a separator, and added an additional comma if not all pieces were removed (to handle the edge case where the input string is empty).
