I approached this problem by iterating through the input list and keeping track of the current number and its count. When a new number is encountered, I check if the count of the previous number is greater than 1, indicating a run of consecutive numbers. In this case, I add that range to a list of runs. After processing all the numbers, I convert each run into a string format and join them with commas, handling the edge case where a single number doesn't require a range by checking its length. If the list is empty, I return an empty string.
