The function `range_summary` first checks if the input list is empty and returns an empty string if true. It then sorts the input list to handle consecutive numbers correctly. The function iterates through the sorted list, identifying ranges of consecutive numbers by checking if the current number is exactly one more than the last number in the range (`end`). When a non-consecutive number is found, it appends either a single number or a range ('start-end') to the `ranges` list, depending on whether there was a single number in the range. After processing all numbers, it finalizes the last range and joins all ranges into a single string separated by commas. This approach ensures that consecutive numbers are collapsed into ranges while handling edge cases like an empty input or a list with only one distinct number.
