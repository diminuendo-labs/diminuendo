Next, if the length of the list modulo 'k' (denoted as len(lst)%k) is not zero, which means there is a remaining final chunk that comes up short, I pad it with None values and add these padded chunks to the result list. This approach ensures that every resulting chunk has exactly 'k' elements, except for an empty list, which returns [].
