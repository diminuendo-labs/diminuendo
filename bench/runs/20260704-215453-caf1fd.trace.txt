I used Python's built-in sort function to arrange the input list in ascending order. Then, I calculated the middle index of the list, considering edge cases where the list size is even. After that, I defined the indices for the minimum and maximum values (ignoring one occurrence each). Finally, I calculated the sum of the numbers at those indices, divided by the total number of elements minus 2 to find the trimmed mean. The function returns None if the input list is empty.
