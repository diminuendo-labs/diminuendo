I chose to use a straightforward approach to solve this problem. I first split the input string into words, converting them to lowercase. Then, I used Python's built-in `Counter` class from the `collections` module to count the occurrences of each word. Since we only need the most common word(s), I found the maximum count and filtered the words with that count. To break ties alphabetically, I sorted the list of most common words in ascending order and returned the last one (index -1). This approach is concise and readable, and it handles the edge case where the input string is empty.
