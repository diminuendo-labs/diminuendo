I created a recursive function `flatten_once()` that takes a list as input and iterates through each item. If the current item is itself a list, it calls the same function (recursively) on this inner list to flatten it further before appending its elements to the final result. Otherwise, it simply adds the non-list element to the result. This approach allows for one level of nesting to be flattened while preserving deeper nesting and handles edge cases where there might be lists within lists at any depth in the given list.
