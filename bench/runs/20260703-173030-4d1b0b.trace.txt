I used a list comprehension to iterate through each element in the input list. If an element is not a list, it's kept as it is. Otherwise, recursively call the `flatten_once()` function on the nested list. The edge case I handled is when there are no more layers of nesting within a sub-list, and the sub-lists only contain primitive data types like integers, floats, or strings. In such cases, these sub-lists will be flattened and joined with the original list.
