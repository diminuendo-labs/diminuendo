I implemented the function `max_index` to find the index of the largest value in a list, returning the last occurrence when it appears multiple times. The approach involves iterating through the list with an enumeration to keep track of both the index and value. If the current number is greater than or equal to the currently stored maximum value (`max_val`), I update `max_val` and set `max_index` to the current index. This ensures that if there are multiple instances of the maximum value, the last one's index will be returned. I also added an edge case check for an empty list, raising a `ValueError` if such a case occurs.
