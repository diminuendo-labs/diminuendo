To achieve this, I first split the input string by underscores to get a list of words. Then, I initialize `camelCase` with the lower case version of the first word, and iterate through the remaining words (excluding the first one), capitalizing each one and appending it to `camelCase`. Finally, I return the resulting camelCase string or an empty string if the input was empty.

For handling the edge case where there are leading, trailing, or repeated underscores, splitting by '_' automatically drops extra underscores. The first and last underscores don't affect words in `words`, so they won't be included in the final camelCase string.
