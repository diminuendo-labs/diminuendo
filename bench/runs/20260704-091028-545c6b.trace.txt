In this solution, the function `median()` takes a list of numbers and returns the middle value as a float in case of an even number of elements or the single middle value when the length is odd. The list is first sorted for accurate results. To handle an edge case where the input list is empty, we return None. The middle index is calculated by integer division, which automatically handles the situation when the length is even by rounding down for the first middle index and up for the second (in case of a tie). If the length is even, the two middle values are averaged to find the median.
