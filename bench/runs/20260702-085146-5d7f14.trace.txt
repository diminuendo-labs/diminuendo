The function `flatten_once` takes a list as input and iterates through each element. If the element is another list, it uses the `extend` method to add its elements to the result list in order. Otherwise, it simply appends the non-list element to the result list. This approach ensures that only one level of nesting is flattened, while deeper levels remain nested. The edge case handled here is an empty input list, which will return an empty result list as expected.
