I used Python's string methods to achieve the desired result. The `split()` method breaks the input string into words, and I iterate over each word. If a word is entirely uppercase, it's added to the result list without change. For other words, I use `title()` to convert only the first letter to uppercase and the rest to lowercase. Finally, I join the processed words back together with spaces using `' '.join()`. The edge case of a single uppercase word ('ABC') is handled by checking if the word is entirely uppercase before applying title casing.
