I approached the problem by first ensuring that the input list is sorted and free of duplicates. This is achieved using `sorted(set(nums))`. The function then iterates through this list, identifying consecutive sequences. When a sequence ends, it checks whether the start and end of the sequence are the same. If they are, it appends a single number; otherwise, it appends the range in 'start-end' format. Finally, the results are joined into a comma-separated string and returned. The edge case of an empty list is handled by returning an empty string at the beginning of the function.
