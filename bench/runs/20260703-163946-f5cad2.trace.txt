The function `range_step(start, stop, step)` takes in three arguments and returns a list of integers between `start` (inclusive) and `stop` (exclusive), advancing by `step`. If `step` is zero or negative, the function returns an empty list. To handle the edge case where the `current` value equals `stop`, we pop the last element from the list if it occurs. The code uses a simple while loop and appends each valid result to the list. To make the code concise and readable, I opted for explicit variable names and used a separate while loop instead of one-liners.
