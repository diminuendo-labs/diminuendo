To solve this task, I first check if the input list is empty and return an empty string if true. Then, I sort the list to handle consecutive numbers easily. I use two pointers (`start` and `end`) to track the current range. As I iterate through the sorted list, if the current number is consecutive to the end of the current range, I update the end pointer. If not, I append the current range (or single number) to the result list and reset the start and end pointers to the current number. After the loop, I add the last range to the result. Finally, I join the result list into a string with commas separating the ranges and return it. This approach ensures that consecutive runs are collapsed into 'start-end' format while single numbers remain as is.
