I approached this task by first removing all underscores from the given string and then splitting it into separate words using the split() function. This helped in dealing with multiple consecutive underscores. To ensure that the first letter of each word was properly capitalized, I used the map() function with a lambda expression to apply the capitalize() method on each word. Lastly, to account for the edge case where the input string is an empty string or contains only underscores, I handled it by returning an empty string in such cases. This keeps the solution concise and readable while efficiently handling multiple edge cases.
