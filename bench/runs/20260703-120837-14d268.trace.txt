The function `range_summary` first checks if the input list is empty and returns an empty string if it is. It then sorts the unique elements of the list to ensure they are in ascending order. The main logic uses a while loop to iterate through the sorted numbers, identifying consecutive sequences by checking if each number is exactly one more than the previous. When a sequence ends (or when the end of the list is reached), it checks if the sequence length was 1 (i.e., just one number) or greater (indicating a range). The function formats these accordingly and appends them to a summary list. Finally, it joins all parts of the summary with commas to produce the desired output string.
