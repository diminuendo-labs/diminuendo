I started by checking for the edge case of an empty list, returning an empty string if present. I then sorted and deduplicated the input list to ensure all numbers are in ascending order without duplicates. Using a loop, I identified consecutive ranges by comparing each number with the previous one. If a range was found (i.e., the current number is exactly one more than the last), I updated the end of the range. When a non-consecutive number was encountered, I appended the current range to the result list in the appropriate format ('start-end' or just 'number'). Finally, I joined all ranges into a single string with commas and returned it.
