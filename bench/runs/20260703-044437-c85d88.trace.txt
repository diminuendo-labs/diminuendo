To solve this task, I used the `collections.Counter` class to count occurrences of each word. After determining the maximum count, I filtered words that match this count and sorted them alphabetically. The function returns the last word in the sorted list (i.e., the lexicographically largest one) to adhere to the tie-breaking rule specified in the task. This approach ensures correctness, conciseness, and readability while handling the edge case of an empty string input gracefully.
