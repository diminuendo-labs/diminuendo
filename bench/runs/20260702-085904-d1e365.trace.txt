**  
The function `most_common_word(s)` first checks if the input string `s` is empty and returns an empty string if true. It then converts the entire string to lowercase and splits it into individual words. A dictionary, `word_count`, is used to count occurrences of each word. After counting, it finds the maximum count and selects all words that have this highest frequency. Finally, it sorts these most common words alphabetically in ascending order and returns the last one, which ensures breaking ties alphabetically as required. This approach handles both the edge case of an empty string and correctly processes cases where multiple words tie for most occurrences.
