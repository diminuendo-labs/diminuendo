I used the `Counter` class from the `collections` module to count the occurrences of each word in the string. However, since we need to break ties alphabetically, I opted for a simpler approach using the `max` function with a custom key. The key is a tuple where the first element is the count of the word and the second element is the word itself. This ensures that words with higher counts come first, and in case of a tie, the one that comes alphabetically first is chosen.
