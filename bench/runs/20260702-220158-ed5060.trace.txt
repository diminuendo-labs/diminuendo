**
The function `title_words(s)` processes the input string `s` by first splitting it into individual words. It then iterates over each word, checking if it is entirely uppercase using `word.isupper()`. If a word is entirely uppercase, it is added to the result list as is; otherwise, the word is title-cased using `word.title()` and added to the result list. Finally, the function joins the processed words back into a single string with spaces in between and returns this string. The edge case of an empty string is handled by immediately returning an empty string if the input is empty.
