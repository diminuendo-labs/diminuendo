The function takes a string `s`, splits it by spaces and applies a helper function `titlecase` to each word. The helper function checks if the first letter of a word is uppercase or if the word is empty. If either condition is true, the word remains as is; otherwise, the first letter becomes uppercase, and the rest lowercase. Finally, the function collapses multiple spaces into single spaces by using `' '.join()`.

The edge case handled here is when the input string contains words that are already entirely uppercase (e.g., "ACADEMY", "HTTPS"), which remain unchanged in the output.
