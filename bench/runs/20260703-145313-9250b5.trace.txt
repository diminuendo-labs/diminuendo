I implemented the function using a dictionary to count the occurrences of each number in the input list. This allows for efficient lookup and updating of the counts as we iterate through the list. The outer loop iterates over each number in the list, calculating its complement with respect to the target sum. If this complement has been seen before (i.e., it's a key in the dictionary), we increment our pair count by the number of times it has appeared; otherwise, we add 1 to the count for this number. The edge case where an empty input list is handled at the beginning of the function.
