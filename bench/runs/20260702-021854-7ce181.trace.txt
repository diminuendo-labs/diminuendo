I chose to use a list comprehension to create a new list that includes all elements from the input list. The outer comprehension iterates over each element in the input list. If an element is itself a list (checked using `isinstance(sublist, list)`), its items are added to the result list. If not, the original element is added as a single-element sublist. This approach is concise and readable, and it handles edge cases like empty lists or non-list elements.
