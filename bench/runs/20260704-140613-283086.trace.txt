The function `trimmed_mean` first checks if the length of the input list `nums` is less than or equal to 2. If so, it returns `None` as specified for edge cases involving lists that are too short to perform the trimming operation. If the list has more than two elements, it removes one occurrence each of the minimum and maximum values from the list using the `remove` method. Finally, it calculates and returns the mean of the remaining numbers as a float by summing them up and dividing by their count.
