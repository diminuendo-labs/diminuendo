I solved this task by first sorting the input list using a lambda function as the key. This key returns a tuple for each word, where the first element is the length of the word and the second element is the word spelled backwards (case-insensitive). The sorted function then uses these tuples to determine both the order within groups of words of equal length and the order between different length groups. I chose to adjust `k` to a 0-indexed position before attempting to access the list, as this simplifies the try-except block needed to handle out-of-range values. The edge case where the input list is empty is explicitly handled at the beginning of the function.
