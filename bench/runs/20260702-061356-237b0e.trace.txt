I defined the `flatten_once` function to iterate over each element in the input list. If an element is a list itself, it's unpacked and added to the result list; otherwise, the element is added as-is. This ensures that only one level of nesting is flattened. The function uses list comprehension for conciseness. I also included a simple test function to verify the correctness of `flatten_once` with various edge cases, ensuring it handles lists within lists, all elements being lists, and no nested elements correctly.
