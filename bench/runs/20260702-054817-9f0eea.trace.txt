**

The function `balanced(s)` checks whether the parentheses in a given string are balanced. It uses a stack to keep track of the opening parentheses encountered. For each character in the string, if it's an opening parenthesis '(', it is pushed onto the stack. If it's a closing parenthesis ')', the function checks if there is a corresponding opening parenthesis at the top of the stack; if not, or if the stack is empty before finding a match, the function returns `False`. After processing all characters, if the stack is empty, it means all parentheses were properly closed, and the function returns `True`. The edge case handled is an empty string, which is considered balanced as there are no unmatched parentheses.
