I split the input string by underscores to isolate each word. I then iterate over these words, capitalizing them unless they're in the exception list, where they remain unchanged. The first token is converted to lowercase and concatenated with the rest of the tokens. This results in a camelCase string. One edge case this function handles is when the input string starts with an underscore, as the split operation will result in an empty first token which is ignored when constructing the final string.
