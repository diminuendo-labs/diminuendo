I created a function that takes two lists as arguments, `values` and `weights`, where each value corresponds to its respective weight. The edge case I addressed is when either of the input lists are empty or the total weight (sum of weights) is zero, in which case the function returns None instead of throwing an error. To calculate the weighted mean, I used a list comprehension to iterate through both lists and perform multiplication before summing up the results. Finally, I divide by the total weight to get the average value. The code is concise and readable for easy understanding.
