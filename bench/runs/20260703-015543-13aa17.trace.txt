I approached this task by first splitting the input string into a list of whitespace-separated words using `str.split()`. Then, I used the built-in function `max()` with a generator expression to find the length of the longest word. To handle ties and return the last such word, I reversed the list of words before finding the maximum length, and then used another generator expression in combination with `next()` to return the first word (i.e., the last one in the reversed list) that has this maximum length. The edge case where an empty string is passed as input is handled implicitly by returning an empty string when there are no words.
