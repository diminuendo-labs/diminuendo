To achieve the desired camel case conversion while handling exceptions, I split the input string by underscores into a list of tokens. The first token is converted to lowercase immediately. For each subsequent token, if it's not in the exception list, it's capitalized; otherwise, it remains unchanged. Finally, all tokens are joined together without any separators to form the final camel case string. This approach ensures that the function correctly handles edge cases involving specific identifiers like 'URL' and 'ID'.
