I implemented a solution using the Stack data structure to keep track of opening parentheses in the string. Each time an opening parenthesis is encountered, it gets pushed onto the stack. Whenever a closing parenthesis appears, if the stack is empty or the top item on the stack is not an opening parenthesis, the function returns False indicating that the parentheses are not balanced. If the function completes without returning False, and the stack is empty, the parentheses are balanced, so the function returns True. I chose this approach because it's efficient, straightforward, and easy to understand. One edge case handled is when there are unmatched or mismatched parentheses within the input string.
