The function `median` first checks if the input list `nums` is empty and returns `None` if it is. If not, it sorts the list in ascending order. Depending on whether the length of the sorted list `n` is odd or even, it calculates the median differently: for an odd length, it returns the middle element; for an even length, it calculates the average of the two middle elements. This approach ensures that the function handles both cases correctly and efficiently.
