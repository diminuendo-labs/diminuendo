I designed the `flatten_once` function to iterate through each element in the input list. If an element is a list itself, it extends the result list with its items, effectively flattening one level of nesting. For non-list elements, it appends them directly to the result list. This approach handles the edge case where the input list contains sublists that are not lists themselves (e.g., `flatten_once([1, [2], 3])` should return `[1, 2, 3]`, not raise an error).
