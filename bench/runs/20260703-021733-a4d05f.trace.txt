The function `snake_to_camel` takes a string `s` and converts it from snake_case to camelCase. It first strips any leading or trailing underscores using `strip('_')`. Then, it splits the string into words based on underscores, filtering out any empty strings that might result from consecutive underscores. Each word is then converted to title case unless it's already entirely uppercase (to preserve special cases like 'JSON' which should remain unchanged). The resulting list of words is joined together without separators to form the final camelCase string.
