We first handle the edge case of an empty or single-element list by returning None. Then, we sort the input list and check for cases where both minimum and maximum values have more than one occurrence. If neither case is true, we calculate the mean of all but the first and last elements (excluding them since they're either the minimum or maximum). If one or both of the cases are true, we remove the corresponding element(s) from the list and recalculate the trimmed mean. The resulting function returns a float representing the trimmed mean of the list.
