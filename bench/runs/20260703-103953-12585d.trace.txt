I approached this task by first checking for the edge case where the input string is entirely whitespace or empty. If so, I return an empty string immediately. For non-empty strings, I split the string into words using `split()`, which handles multiple spaces correctly. Then, I use Python's built-in `max()` function with the `key=len` argument to find the longest word. Since `max()` returns the first item with maximum length in case of a tie by default, and the problem specifies that we should return the last such word, no additional logic is needed for handling ties.
