**  
The function `chunk_pad` is designed to split a given list into consecutive chunks of size `k`. If the final chunk does not have exactly `k` elements, it pads the chunk with `None` values. The solution uses list comprehension to iterate through the original list in steps of `k`, slicing out each chunk. For each slice, if its length is less than `k`, it appends the difference as `None` values to reach a total length of `k`. This ensures that every chunk has exactly `k` elements. The edge case where an empty list is passed is handled gracefully by returning an empty list without any padding.
