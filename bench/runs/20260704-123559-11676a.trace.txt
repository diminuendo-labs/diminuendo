I created a function `trimmed_mean(nums)` that takes a list of numbers as an input. If the list is empty, it returns `None`. If the length of the list is 1, it simply returns the single number as the mean. Otherwise, I sort the list in ascending order using the `sort()` method, then remove the first and last elements (minimum and maximum) from the list by slicing `nums[1:-1]`. Finally, I calculate the mean by summing the remaining numbers and dividing by their count. This handles an edge case where the input list contains only one unique number.
