I first checked if the input list `nums` is empty and returned an empty string if it is. Then, I sorted the list to ensure that consecutive numbers are adjacent. I used a loop to iterate through the sorted list, tracking the start and end of each run of consecutive numbers. When a non-consecutive number is encountered, I added the range (or single number) to the result list in the required format. Finally, I joined the result list into a string with commas and returned it.

This approach ensures that all distinct values are included and runs are collapsed correctly. The edge case of an empty list is handled by immediately returning an empty string.
