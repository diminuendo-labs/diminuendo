The function `merge_ranges` takes a list of pairs representing [start, end] ranges and first sorts them by their starting points using the built-in Python sort function. Next, it iterates through each pair in the sorted list. If the current pair doesn't have a previous range or the end of the previous range is less than the start of the current pair, then we append the current pair to the result list. If there is a previous range and its end is greater than or equal to the start of the current pair, then we merge the ranges by updating the end of the previous range with the maximum of its current end and the end of the current pair.

The edge case that this function handles is when the input list contains overlapping or touching ranges, and it merges them into a single merged range as per the problem statement.
