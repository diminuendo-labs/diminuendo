The approach involves sorting the list of numbers in ascending order and calculating the index of the median. Then, we find the indices of the minimum and maximum values while skipping over any repeated occurrences at the start or end. Finally, we sum up all numbers between the first (excluding the minimum) and the last skipped maximum value, excluding one more value if the minimum is the first element in the list. The total count of numbers to be averaged is adjusted accordingly, and we return the trimmed mean as a float. This solution handles an edge case where the input list contains only duplicate minimum or maximum values.
