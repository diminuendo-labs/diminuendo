I defined a function `count_token(s, w)` which takes two arguments, string `s` and target word `w`. I first split the input string `s` into words using space as a delimiter. To handle leading and trailing punctuation from each word, I used the `strip('.,!?:;')` method on each word before adding it to a set (to eliminate duplicate words).

Then, I created another set with all lowercase versions of the cleaned words. Since sets do not contain duplicates, I subtracted 1 from the result when comparing the target word `w` with the words in the new set because one count would have been added for each occurrence. If the target word `w` is an empty string, the function returns 0. I also included an edge case where a duplicate target word (ignoring case) will not be counted twice.
