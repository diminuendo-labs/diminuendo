I chose to approach this problem by first removing duplicates from the input list and sorting it, ensuring that I can efficiently iterate over consecutive numbers. Then, I used two variables `start` and `end` to track the range of consecutive numbers. If a number is not part of a consecutive run (i.e., there's a gap), I append the previous range to the result list and reset `start` and `end`. This way, I can generate the desired string in a single pass over the input list. The code is concise and readable due to the use of Python's built-in functions like `set`, `sorted`, and f-strings for formatting the output.
