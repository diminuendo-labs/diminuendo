The function `snake_to_camel` takes a string `s` as input. It first strips any leading and trailing underscores using `strip('_')`. Then, it splits the string by underscores with `split('_')`, which gives us a list of words. We use a generator expression within `join()` to capitalize each word and concatenate them into one string. If a word is empty (which can happen with multiple consecutive underscores), it is skipped. This solution handles leading, trailing, and repeated underscores effectively, and an empty string returns an empty result.
