The function `equal_ignoring_spaces` takes two strings as input and returns `True` when both strings are equal after removing spaces and converting them to lowercase. It uses the built-in `filter()` function with `str.isalnum()` to filter out non-alphanumeric characters (whitespaces), then concatenates the resulting filtered sequence into a single string using the join method. To handle whitespace characters, we're only considering alphanumeric ones in our filter. Lastly, both strings are converted to lowercase before comparing them for equality. An edge case considered here is when either of the input strings could be empty (both null or non-null), in which case they are considered equal if that condition holds true.
