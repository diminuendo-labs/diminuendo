The function `median` takes a list of numbers as input and returns the median value. It first checks if the list is empty and returns `None` if it is. If not, it sorts the list in ascending order. The length of the sorted list is then calculated, and the middle index is determined. If the list length is odd, the function returns the value at the middle index. If the list length is even, it calculates the average of the two middle values and returns it as a float. This approach ensures that the median is correctly computed for both odd and even lengths of the input list, handling the edge case where the list might be empty.
