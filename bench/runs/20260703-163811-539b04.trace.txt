I created a Python function named `weighted_mean()` that takes two arguments - `values` and `weights`. The function first checks if the input lists are empty or have a zero total weight, in which case it returns `None`. Then, it calculates the sum of the weighted values using a generator expression and computes the weighted arithmetic mean by dividing the total sum by the sum of weights. An edge case handled is when both the input lists are empty or have a zero total weight.
