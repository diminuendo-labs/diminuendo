I chose to approach this task by first splitting the input string into a list of words using the `split()` method. Then I found the maximum length of any word in the list using the `max()` function along with the `map()` and `len()` functions. After finding the maximum length, I used a generator expression (`(word for word in words if len(word) == max_len)`) to iterate through all words of that maximum length. Finally, I used the `next()` function to return the first matched word found or an empty string if no such word is present. The use of a generator expression allows me to follow the stated direction of returning the last occurrence when there are ties, as it iterates over words in their order of appearance.
