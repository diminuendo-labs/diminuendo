I used the split() method to separate the string into a list of words. Then I applied the built-in function max() with a custom key function (key=len), which will sort the list based on the length of each word in descending order. If the input string is empty, an empty string ('') will be returned. To handle an edge case, if the longest words have the same length, it returns the last such word, not the first one as required by the task description.
