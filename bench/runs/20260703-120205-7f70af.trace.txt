The solution involves splitting the input string `s` into a list of words using `split()`. It then checks if the list is empty to handle the edge case where the input string is empty. Using the built-in `max()` function with `len` as the key, it finds the longest word. Since `max()` returns the first longest word in case of a tie by default, this naturally handles the requirement to return the last such word when there's a tie.
