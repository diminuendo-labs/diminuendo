I started by checking if the input string `s` is empty. If it is, I return an empty string as per the edge case handling. Next, I convert the entire string to lowercase and split it into individual words. I then count the occurrences of each word using a dictionary. Finally, I use the `max()` function with a custom key that first considers the negative count (to get the highest counts) and then alphabetically for ties. This ensures that if there are multiple words with the same highest count, the lexicographically smallest one is returned.
