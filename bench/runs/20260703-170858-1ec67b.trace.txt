I created a function `second_largest(nums)`. The approach involves first converting the input list to a set to remove any duplicates. Then, I sorted the set in descending order. If there are fewer than two distinct values (i.e., the list contains only one value), the function returns `None`. Finally, it returns the second largest distinct value by accessing the last but one element from the sorted set. The edge case handled is when there is only one unique value in the list.
