First, I used the `split()` method to separate the input string `s` into a list of words by whitespace. Then, I applied a generator expression to iterate over each word and compare it case-insensitively with the input word `w`. Finally, I used the `sum()` function to count the number of times the comparison evaluates to True.

The edge case that I've handled is when `s` is an empty string; in this situation, `count_word("", "")` will return 0 instead of raising an error.
