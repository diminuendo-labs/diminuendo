I implemented the function `median` that takes a list of numbers as input. To calculate the median, I first check if the input list is empty and return None in this case. Then, I sort the list using the built-in `sorted` function. If the length of the sorted list is even, I return the average of the two middle values by adding them together and dividing by 2. If the length is odd, I simply return the middle value. This approach is concise and readable while handling an edge case where an empty list returns None.
