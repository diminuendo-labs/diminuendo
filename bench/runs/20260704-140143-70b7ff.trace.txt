**

The function `trimmed_mean` calculates the mean of a list after removing one occurrence of both the minimum and maximum values. If the list is empty, has only one element, or has two elements, it returns `None`. The function uses Python's built-in `min` and `max` functions to find these values and then removes their first occurrences using the `remove` method. Finally, it calculates and returns the mean of the modified list as a float.
