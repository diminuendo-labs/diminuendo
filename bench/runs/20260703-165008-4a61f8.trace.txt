The function first splits the input string into a set of words (to remove duplicates), and then it counts the frequency of each word using a dictionary. The minimum frequency and corresponding least frequent word are found by using the `min()` function with a custom key function, which considers both the frequency and the word itself as a tuple.

An edge case that the function handles is when the input string contains no words (i.e., only whitespace characters). In this situation, the function returns an empty string instead of raising an exception or returning a special value like None.
