The function `collapse_delims()` first defines the set of delimiters it will handle (commas, semicolons, and spaces). It then splits the input string using these delimiters. To account for the edge case where a single delimiter appears at the beginning or end of the string without any content, the function checks if `pieces` is empty and if the last character of the original string is one of the delimiters. If so, it adjusts the pieces list accordingly (either by removing the first character or the last character). Finally, it removes any empty pieces from the list using the built-in `filter(None, ...)` function and rejoins the remaining pieces with a single comma. The resulting string is then returned. This approach ensures that consecutive delimiters are treated as one split, leading to the desired output.
