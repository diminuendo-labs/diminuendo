In this solution, we traverse the input string from left to right and maintain two variables: `depth` to keep track of the current nesting level, and `max_depth` to store the maximum depth encountered so far. If a '[' is found, we increment the `depth`. If a ']' is found, we decrement the `depth` and check if the `depth` is 0 (indicating an unmatched opening bracket). If there's ever a closing bracket before its corresponding opening one, we return -1 as an edge case. Finally, we return the maximum depth encountered or 0 for an empty string.
