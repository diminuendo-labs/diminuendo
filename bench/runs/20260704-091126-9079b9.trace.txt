I used the built-in Python function `sort()` to sort the input list of numbers. Then, I calculated the middle index of the sorted list by taking the floor division and checking if the length is odd or even using modulo operation (`%`). If the length is odd, I returned the value at the middle index; otherwise, I found the average of the two middle values. The edge case handled in this solution is when the input list is empty, in which case the function returns `None`.
