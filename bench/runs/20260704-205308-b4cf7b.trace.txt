The function first checks if the list is empty and returns None in that case. Then it sorts the list and calculates the indices for the minimum, maximum, and median values. The trimmed mean is calculated as the average of all elements except the smallest (excluding one occurrence) and the largest (also excluding one occurrence). If the length of the list is less than 4, we adjust the number of elements to exclude by 2 instead of 4 to handle edge cases when the length is exactly 3.
