The function `snake_to_camel` takes a string in snake_case format and converts it to camelCase. It first strips any leading or trailing underscores using `strip('_')`. Then, it splits the string by underscores into parts using `split('_')`. For each part, if the part is in lowercase (`islower()`), it converts it to title case using `title()`, which capitalizes the first character and makes the rest lowercase. If the part is already uppercase or contains mixed cases, it leaves it unchanged. Finally, it joins all parts together without any separators using `''.join(...)`. This approach ensures that the conversion handles leading, trailing, and repeated underscores correctly while respecting the task's requirement to not apply title casing to words that are already entirely uppercase.
