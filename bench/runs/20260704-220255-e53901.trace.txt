I created a function called `trimmed_mean` that takes in a list of numbers. If the input is an empty list, it returns None. I used the Python built-in `sort()` method to sort the list in ascending order. After that, I defined variables for the indexes of the minimum and maximum values, as well as the number of elements left to calculate the mean after removing one occurrence of each extreme value. Lastly, I calculated the trimmed mean by summing the middle (non-extreme) values in the sorted list and dividing it by `n`, and then rounding the result to 2 decimal places before returning it. The edge case I named was an empty input list.
