I chose to use a stack to keep track of the opening parentheses. As I iterate through each character in the string, I push an opening parenthesis onto the stack. When I encounter a closing parenthesis, I check if the stack is empty or the top of the stack is not an opening parenthesis; if either condition is true, the parentheses are unbalanced, and I return `False`. If everything checks out, I pop the opening parenthesis from the stack. Finally, after iterating through the string, if the stack is empty, it means all parentheses were balanced, and I return `True`; otherwise, I return `False`.

This approach ensures that only valid pairs of parentheses are considered, and it efficiently handles edge cases such as an empty string or strings with unbalanced parentheses.
