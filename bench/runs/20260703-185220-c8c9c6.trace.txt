I chose to use the built-in `split()` method to separate the string into a list of words. Then, I found the length of each word in the list and stored it in a variable `max_len`. The max_len is used as the key to compare the lengths of all words. If two or more words have the same maximum length, the last one will be returned because it has the greater index in the list (using float('-inf') for comparison). The edge case I handled is when the input string is empty; in that case, the function returns an empty string instead of raising an exception.
