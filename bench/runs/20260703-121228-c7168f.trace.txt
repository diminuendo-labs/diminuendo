To implement the `longest_word` function, I first check if the input string `s` is empty. If it is, the function returns an empty string as specified in the edge case. Otherwise, I split the string into words using the `split()` method, which by default splits on whitespace. I then use the `max()` function with the `key=len` argument to find the longest word in terms of length. In case of a tie (multiple words of the same maximum length), `max()` returns the last such word due to Python's stability sort behavior. If there are no words in the list, `max()` returns an empty string thanks to the `default=''` parameter. This approach ensures the function is both concise and readable while correctly handling edge cases.
