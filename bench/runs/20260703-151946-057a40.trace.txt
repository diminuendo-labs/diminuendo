**  
The function `range_summary` takes a list of integers, removes duplicates by converting it to a set and sorting the unique values. It then iterates through these sorted numbers to identify consecutive sequences. If a sequence is found (i.e., the current number is exactly one more than the previous), the end of the range is updated. When a non-consecutive number is encountered, the function checks if there was an existing range and appends it to the result list in the appropriate format ('start-end' or just 'number'). Finally, any remaining range is added to the result before joining all parts with commas to form the final output string. The edge case of an empty input list is handled by returning an empty string.
