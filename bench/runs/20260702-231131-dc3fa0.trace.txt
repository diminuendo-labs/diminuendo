I chose to use a set to remove duplicates from the input list, and then sorted the resulting list. This is because the task requires distinct values in ascending order, making it easier to identify consecutive runs of numbers. To handle consecutive runs, I iterate through the sorted list, comparing each number with its successor. If they are consecutive (i.e., their difference is 1), I append a range to the result list; otherwise, I append either the current number or a new range. The edge case of an empty input list is handled by returning an empty string immediately at the beginning of the function.
