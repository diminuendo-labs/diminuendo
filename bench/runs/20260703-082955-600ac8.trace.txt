I chose to use Python's built-in `Counter` class from the `collections` module to efficiently count the frequency of each number. This approach allows me to avoid iterating over the list multiple times and directly get the maximum frequency. I also explicitly handled the edge case where the input is an empty list, as it is a crucial consideration for this problem. By using a list comprehension to filter the distinct numbers with the maximum frequency, I aimed to make the code concise while still being readable.
