To implement the `median` function, I first check if the input list is empty and return `None` if it is, handling the edge case of an empty list. Then, I sort the list to arrange the numbers in ascending order. Depending on whether the length of the sorted list is odd or even, I calculate the median accordingly: for an even number of elements, I take the average of the two middle numbers; for an odd number, I return the middle number directly as a float. This solution ensures correctness and readability while addressing the edge case of empty input.
