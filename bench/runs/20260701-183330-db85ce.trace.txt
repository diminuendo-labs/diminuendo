To solve this problem, I first created a dictionary `char_count` to store the count of each character in the string. By iterating over the string and updating the counts, I was able to efficiently determine which characters appear more than once. Then, I iterated again through the string to find the first character with a count of one, returning its index. If no such character exists, the function returns -1. This approach ensures that all characters are counted only once, making it efficient even for longer strings.
