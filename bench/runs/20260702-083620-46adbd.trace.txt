The function `average` calculates the arithmetic mean of a list of numbers. It first checks if the input list `nums` is empty, in which case it returns `None`. If the list is not empty, it computes the average by dividing the sum of the list elements (`sum(nums)`) by the number of elements in the list (`len(nums)`). This ensures that even if the mean is a whole number, it is returned as a float. The function handles the edge case where an empty list should return `None`, avoiding division by zero or returning an incorrect result.
