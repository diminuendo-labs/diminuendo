First, I defined an exceptions list that contains the words 'URL' and 'ID', to keep their exact form wherever they appear. Then, I split the underscore-delimited identifier into a list of tokens. For each token, I check if it is not empty, starts with an alphabet character (excluding exceptions), and has an underscore as its separator. If these conditions are met, I lowercase the first letter and capitalize the rest of the letters for that token, otherwise, the token remains unchanged. Lastly, I join all the transformed tokens together to create a single string in camel case.
