**  
The function `range_summary` takes a list of integers, sorts it to ensure they are in ascending order, and then iterates through the sorted numbers to identify consecutive runs. It uses two pointers, `start` and `end`, to track the beginning and end of each run. When a non-consecutive number is encountered, it appends either a single number or a range ('start-end') to the result list. After the loop, it handles the last range or number in the list. The function returns the concatenated string representation of all ranges or numbers separated by commas. An empty input list results in an immediate return of an empty string.
