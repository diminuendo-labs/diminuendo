To implement the `chunk_pad` function, I decided to use a list comprehension that iterates through the input list `lst` in steps of size `k`. For each iteration, it slices the list from the current index `i` up to `i+k`, which gives a chunk. If this chunk is shorter than `k`, it pads the chunk with `None` until its length equals `k`. This ensures that every chunk has exactly `k` elements. The edge case where the input list is empty is handled by returning an empty list directly, as no padding or chunking is needed in this case.
