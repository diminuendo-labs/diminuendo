First, I defined a set of exceptions where the tokens should keep their exact form. Then, I converted the underscore-delimited string to a space-delimited string and made the first letter lowercase. After that, I capitalized every subsequent word (space) and removed spaces. Finally, I checked if the first token was in the exceptions set and replaced it with the exact exception token if true. This approach handles edge cases where an exception token is the first token of the identifier.
