The function `range_summary` takes a list of integers and returns a string representing the range summary. It first checks if the input list is empty, returning an empty string if so. The list is then sorted to ensure that consecutive numbers are adjacent. A loop iterates through the sorted list, tracking the start and end of each sequence of consecutive numbers. When a break in the sequence is detected, the current sequence (or single number) is added to the result list in the required format ('start-end' or 'single_number'). Finally, any remaining sequence is appended before joining all parts with commas to form the final result. This approach ensures that each distinct range or number is represented correctly and efficiently.
