I created a function named `balanced` that takes a string as input. The idea is to iterate through the characters of the string using a for loop. If a '(' character is encountered, it gets added to a stack. When a ')' character appears, if the stack is empty or the last element in the stack is not '(', the function returns False, indicating that the parentheses are unbalanced. In all other cases, the function checks if the stack is empty at the end of the iteration, meaning there are no unclosed parentheses left, and returns True to indicate a balanced set of parentheses. I handled one edge case where an empty string is considered balanced.
