The function iterates through each element in the input list. If an element is a primitive data type like integer, float or string, it is kept as is. However, if an element is a list, it is recursively flattened by calling `flatten_once(i)`. This continues until we reach a level where there's no more nesting. The edge case we handle is when the input list contains both primitives and nested lists with no further nesting (one-level deep).
