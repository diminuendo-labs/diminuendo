I wrote a Python function named `clamp` that takes three arguments - `value`, `lo` and `hi`. If the lower bound is greater than the higher bound, which represents a degenerate range, the function returns `None`. Otherwise, it checks whether the given value falls outside the specified range by comparing with both bounds. In case the value is less than the lower bound or greater than the upper bound, the function clamps the value to the respective limit. If the value lies within the defined range, it simply returns the input value itself. This approach ensures that the function is concise, readable and handles one edge case of a degenerate range.
