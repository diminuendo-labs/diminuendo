I used Python's built-in `max()` function to find the largest value in the list. To handle the case where the largest value appears more than once and we need the index of its last occurrence, I reversed the list with slicing (`nums[::-1]`), found the index of the maximum value in this reversed list (which corresponds to the last occurrence in the original list), and then returned it. This approach ensures that if the input list is empty, the function returns `None`, handling an edge case explicitly.
