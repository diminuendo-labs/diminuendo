**  
The function `most_common_word(s)` processes the input string `s` to find the most common word. It first checks if the string is empty and returns an empty string if so. Then, it converts the string to lowercase and splits it into words. A dictionary `word_count` is used to count occurrences of each word. After that, it finds the maximum count and selects all words with this count. These words are sorted alphabetically, and the first one (which will be lexicographically smallest in case of a tie) is returned as the result. This approach ensures the function is both correct and concise while handling ties by alphabetical order.
