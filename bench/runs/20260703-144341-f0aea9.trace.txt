I approached this task by first checking for the edge case of an empty input list, which would cause a `ValueError` in the subsequent steps. To handle this, I added a simple check with a length condition to see if there are fewer than two distinct values in the list. If so, I return `None`. This approach is concise, readable, and meets all the requirements.
