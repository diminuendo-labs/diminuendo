To achieve the desired output, I first created a list called `runs`. The loop iterates through the input list and checks each number to see whether it's consecutive with the current number. If so, the count is incremented; otherwise, we add the current range (as a tuple of start and count) to our `runs` list and reset the current and count variables for the next run. After iterating through all numbers, I join each run into a single string, using '-' as the separator between the start and end of each range, then ',' as the separator between distinct ranges. If there is only a single number in the input list, it will never be part of a range and will simply be added to the final output string. An edge case I handle specifically is an empty list, which results in an empty output string.
