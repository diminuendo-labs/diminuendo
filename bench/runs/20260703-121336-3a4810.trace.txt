The function `range_summary` first checks if the input list is empty and returns an empty string if true. It then sorts the list while removing duplicates using `sorted(set(nums))`. The main loop iterates through the sorted unique numbers, identifying consecutive sequences by checking if the next number is exactly one greater than the current. If a sequence of length 1 is found, it's added as a single number to the result list. For longer sequences, the start and end of the range are added in the format 'start-end'. The results are then joined into a comma-separated string and returned. This approach ensures that all numbers are included while collapsing consecutive runs into ranges and avoiding any duplicates.
