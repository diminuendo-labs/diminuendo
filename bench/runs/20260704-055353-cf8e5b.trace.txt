I chose a list comprehension approach as it is concise and readable. This function uses the `isinstance` function to check if each element in the input list is also a list. If an element is not a list, it is kept as is in the new list. If an element is a list, only its first element (at index 0) is included in the new list. This approach handles edge cases such as empty lists or single-element lists, ensuring that deeper nesting remains nested.
