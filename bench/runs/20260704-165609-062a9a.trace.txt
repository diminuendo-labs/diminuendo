I defined the function `chunk_pad` that takes a list `lst` and an integer `k`. First, I handle the edge case where the input list is empty or `k` is less than or equal to zero by returning an empty list. Then, I use a list comprehension to create chunks of size `k` from the input list. If the last chunk is shorter than `k`, I pad it with `None` using `[None]*(k-len(lst[i:i+k]))`. This ensures that every chunk has exactly `k` elements.
