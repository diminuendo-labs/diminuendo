The function `range_summary` takes a list of integers, removes duplicates by converting it to a set and then sorting it. It initializes an empty list `result` to store the ranges or individual numbers. Using a while loop, it iterates through the sorted list and identifies consecutive sequences by checking if each subsequent number is exactly one more than the current number. If a sequence of numbers is found, it appends the range in 'start-end' format; otherwise, it appends the single number as a string. Finally, it joins all elements in `result` with commas to form the final output string. The function handles an empty list by returning an empty string immediately.
