The function uses a list comprehension to iterate through each element in the input list. If an element is not a list (meaning it's an integer, float, or string), it remains as-is. Otherwise, the function flattens that sublist by iterating over its contents and appending them to the result list. The edge case we handle is when there are nested lists within other nested lists, as deeper nesting will remain intact since this function only flattens one level of nesting.
