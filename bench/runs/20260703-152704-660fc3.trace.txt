The function first checks if the input list is empty and returns an empty string if true. It then sorts the unique elements of the list in ascending order. Using a while loop, it iterates through the sorted list to identify consecutive runs of numbers. If a run is found (more than one number in sequence), it appends the range 'start-end' to the result list. If not, it appends the single number as a string. Finally, it joins all elements in the result list with commas and returns the resulting string. This approach ensures that consecutive runs are collapsed into ranges while handling edge cases like empty lists or lists with no consecutive numbers correctly.
