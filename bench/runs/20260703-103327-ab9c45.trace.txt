To address the task, I implemented a function `longest_word(s)` that splits the input string `s` into individual words using `split()`. The function then checks for an empty list of words and returns an empty string if true. For non-empty lists, it initializes the longest word as the last word in the list. It iterates through the words in reverse order, updating the longest word whenever a longer or equal-length word is found. This ensures that if there are ties, the last such word is returned. The function finally returns the longest word.
