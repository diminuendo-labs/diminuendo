**
The function `most_common_word(s)` starts by checking if the input string `s` is empty. If it is, it returns an empty string as per the edge case handling. If not, it proceeds to convert the entire string to lowercase and splits it into individual words. Using Python's `collections.Counter`, it counts the occurrences of each word. It then finds the maximum count among all words. Next, it filters out the words that have this maximum count and sorts them alphabetically. Finally, it returns the first word from the sorted list, which is the most common word with ties broken by lexicographical order.
