In this solution, I first sorted the list of words by length and alphabetically within each length group using a lambda function with tuples for combining both criteria. Then, I returned the requested k-th word or `None` if the provided k was out of range. This approach handles the edge case where an empty list is passed, as it would return `None`. Additionally, it maintains readability by utilizing Python's built-in sorting function and clearly defining the ordering criteria using a lambda function.
