**
The function `token_case` takes an underscore-delimited string as input and converts it into camel case. It splits the string by underscores, processes each token according to whether it is in the exception list or not, and joins them back together without any separators. The first token is always converted to lowercase. For other tokens, if they are in the exception list, they retain their original form; otherwise, they are capitalized. This function handles an edge case where tokens listed in the exception (`'URL'`, `'ID'`) remain unchanged regardless of their position in the string.
