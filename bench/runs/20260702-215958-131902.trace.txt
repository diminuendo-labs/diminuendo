The function `weighted_mean` calculates the weighted arithmetic mean of a list of values with corresponding weights. It first checks if either the values list or the weights list is empty or if the total weight is zero, returning `None` in these cases to handle potential errors gracefully without crashing. The main calculation uses a generator expression within the `sum` function to compute the dot product of values and weights, then divides by the total weight to get the weighted mean. This approach ensures that the function is both correct and concise, with clear handling for edge cases such as empty lists or zero weights.
