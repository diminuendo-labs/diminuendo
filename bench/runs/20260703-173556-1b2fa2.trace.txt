Firstly, the function sorts the list of numbers and converts it to a set to remove duplicates. Then, it checks whether the number of distinct values is less than 'n'. If so, it returns None; otherwise, it returns the nth smallest distinct value counting from 1 by indexing it with 'n - 1' as Python uses zero-based indexing. The edge case handled here is when the requested position 'n' exceeds the number of distinct values in the list.
