I used the Python `split()` function to separate the given string into a list of words. Then, I determined the minimum length of any word (or an infinite value `float('inf')` if the list is empty) and used a generator expression with `next()` to find the last occurrence of a word of that specific length. In case there are multiple such words, it will return the last one as per the task description. I handled the edge case where the input string is empty by returning an empty string in this situation.
