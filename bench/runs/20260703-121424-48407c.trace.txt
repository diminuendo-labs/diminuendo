The function `longest_word(s)` splits the input string into words and iterates through them in reverse order. It keeps track of the current longest word encountered by checking the length of each word against the maximum length found so far. If a word is equal to or longer than the max length, it updates the longest word and the max length. By iterating in reverse, if there are multiple words with the same maximum length, the last such word will be returned as required. The function handles the edge case where the input string is empty by returning an empty string immediately.
