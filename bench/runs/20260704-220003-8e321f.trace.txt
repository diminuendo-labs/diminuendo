The function takes a list of numbers as input and returns the trimmed mean. It first checks if the list is empty and returns None in that case. Then it sorts the list and identifies the indices of the minimum and maximum values, excluding one occurrence each. To handle the edge case of an even number of elements, we calculate the average separately for this specific case to ensure that neither the minimum nor maximum value is included in the calculation. Finally, the function returns the calculated average as a float.
