I started by sorting the list and removing duplicates using `sorted(set(nums))` to ensure that each distinct value is only considered once. I then iterated through the sorted list, identifying consecutive ranges by checking if the next number is exactly one greater than the current number. If a range is found, it was formatted as 'start-end'. Single numbers were added as they are. The results of these operations were joined with commas to form the final string. This approach ensures that the function correctly handles the requirements and edge cases, such as an empty list returning an empty string.
