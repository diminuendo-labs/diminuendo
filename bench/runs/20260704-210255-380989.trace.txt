I first sorted the input list in ascending order. Then, I determined the indices of the minimum and maximum values as well as the median index. Since the function requires removing one occurrence of the minimum, one occurrence of the maximum, and the median value is not considered for the calculation, I skipped these indices when summing up the values.
Finally, I calculated the trimmed mean by taking the sum of the remaining values and dividing it by the total number of remaining values (length of the list minus 4). I also added an edge case where the function returns None for an empty list.
