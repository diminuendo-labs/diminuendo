To create the function, I first defined the required inputs and conditions for returning an empty list. Then, I used a generator expression to split the items into consecutive pages of the given size. To handle the edge case when page n exceeds the number of pages but there is still data remaining in the last page, I checked if n is greater than the total number of pages and if the total number of pages is more than 0. In such a case, I returned the last page instead of an empty list.
