To solve this problem, the function `chunk_pad` takes a list `lst` and an integer `k`, representing the desired chunk size. The function uses a list comprehension to iterate through the list in steps of `k`. For each iteration, it slices the list from index `i` to `i+k`, creating chunks. If the last chunk is shorter than `k`, it pads it with `None` using slicing and the expression `[None]*(k-len(lst[i:i+k]))`. This ensures that every chunk has exactly `k` elements. The function handles an empty list by returning an empty list directly, as no padding or splitting is necessary in this case.
