I defined the function `median` that takes a list of numbers as input and calculates its median. To handle the edge case where the input list is empty, I added a condition at the beginning of the function to return `None` in this case. Then, I sorted the input list using the built-in `sort` method (in-place sorting). Finally, I calculated the median based on whether the length of the list is odd or even: if it's odd, I returned the middle value; if it's even, I returned the average of the two middle values.
