I defined a function `token_case(s)` that accepts an underscore-delimited identifier. Inside the function, I first create an exception list containing 'URL' and 'ID'. After that, I split the input string by underscores and iterate through each token. If the token is not in the exception list and it's not empty, I convert its first letter to uppercase and append the remaining letters to `result`. If the token is in the exception list or it's an empty string, I simply append it as-is to the `result`. Finally, I return the resulting camel-cased identifier with a leading lowercase letter by using the title() method.
