I chose to split the input string into individual words using `split()`, which handles multiple consecutive whitespace characters by collapsing them into a single space. For each word, I checked if it was already entirely uppercase with `isupper()`. If it was, I appended it unchanged to the result list; otherwise, I applied title-casing with `capitalize()` (which converts the first character of the string to uppercase and makes the rest lowercase). Finally, I joined the processed words back into a single string with `' '.join(titlecased_words)`, ensuring that only the first letter of each word is uppercased unless it was already all uppercase. This approach maintains readability and conciseness while correctly handling edge cases like multiple spaces between words or words already in title case.
