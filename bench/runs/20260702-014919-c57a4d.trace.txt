I chose to normalize the input string by converting it to lowercase and splitting it into words using `split()`. This approach ensures that the comparison of words is case-insensitive. I then used a dictionary to count the occurrences of each word. To handle ties, I sorted the items in the dictionary first by their count (in descending order) and then alphabetically (in ascending order). Finally, I returned the most common word. The edge case of an empty string was handled at the beginning of the function, ensuring that an empty string is returned immediately if no input is provided.
