To write the trimmed_mean function, I first handle empty and None inputs by returning None. Then, I sort the list to find both the minimum and maximum values easily. The median value is calculated as the middle element of the sorted list. I then filter out one occurrence of the minimum and maximum values from the list using a list comprehension. Finally, the average (mean) of the remaining numbers is calculated and returned as a float. An edge case considered here is when the list has an odd number of elements, where the median is the middle element instead of the average of the two middle elements.
