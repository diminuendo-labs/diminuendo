I created a function called `split_csvish` that takes a string as an argument and returns a list of fields. The function uses the built-in `split()` method to separate the input string by commas, creating a list of individual fields. Then it applies the `strip()` method to each field to remove any leading or trailing whitespace. Finally, it filters out any empty strings from the result using a list comprehension. As for an edge case, the function also handles an input string where all fields are empty (e.g., ", , , "), and it returns an empty list [].
