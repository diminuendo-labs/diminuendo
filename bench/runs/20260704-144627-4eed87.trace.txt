I implemented this function using Python's built-in `sort` method and list comprehension. The key step is sorting the input pairs by their start value, which allows for a straightforward merge process in one pass through the sorted pairs. By iterating over each pair after sorting, I checked for overlaps or shared endpoints with the last merged range; if so, it was updated to reflect the merged interval. If not, the non-overlapping pair was added to the result list.
