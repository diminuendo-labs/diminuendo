I created a function called `flatten_once` that iterates through each item in the input list. If an item is a list itself, it flattens that sublist into individual items; otherwise, it keeps the item as is. This is done using a nested list comprehension where each element is either expanded if it's a list or kept as a single-element list otherwise. The function effectively handles one level of nesting while preserving deeper levels and non-iterable elements intact.
