To write this function, I first removed duplicates by creating a sorted list of unique numbers (using the set() method and then converting it back to a list). Then, I iterated through the list to identify consecutive runs of numbers. Each run was stored as a start-end pair in a list. After the loop, I joined all runs with commas except for single numbers which were left unchanged. For edge cases, this function ensures that each value is counted only once regardless of how many times it repeats in the input list, and it also handles the case where there are no elements in the list by returning an empty string.
