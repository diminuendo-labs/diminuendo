I used a stack data structure to keep track of the opening parentheses. When I encounter an opening parenthesis, I push it onto the stack. When I encounter a closing parenthesis, I check if the stack is empty; if it is, that means there's no matching opening parenthesis, so I return False. Finally, after iterating through all characters in the string, I check if there are any unmatched opening parentheses by checking if the stack is empty; if it isn't, that means there are some unmatched open parentheses and the string is not balanced, so I return False. If the stack is empty, the function returns True.
