I chose to use Python's built-in `split()` function to split the input string into a list of whitespace-separated words. This approach is concise and readable because it leverages existing functionality in the standard library. To handle ties, I used the `key` argument with `len`, which returns the length of each word as a tiebreaker. I also used the `default=''` argument to specify that an empty string should be returned if there are no words (i.e., if the input string is empty).
