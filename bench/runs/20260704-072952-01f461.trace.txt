The function `median` takes a list of numbers as input and returns the median value. If the list is empty, it returns `None`. The list is sorted first to arrange the numbers in ascending order. Depending on whether the number of elements (`n`) is odd or even, the function calculates the median differently. For an odd number of elements, it returns the middle element. For an even number of elements, it calculates the average of the two middle elements and returns it as a float. The edge case of handling an empty list is explicitly addressed by returning `None`.
