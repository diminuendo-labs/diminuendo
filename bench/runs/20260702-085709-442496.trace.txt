** I created a function named `median` that takes a list of numbers as input. If the list is empty, it returns `None`. For non-empty lists, it sorts the list and calculates the median based on whether the length of the list is odd or even. If the length is even, it calculates the average of the two middle numbers; if odd, it returns the middle number directly. This approach ensures that the function correctly handles both even and odd-length lists and avoids unnecessary operations for empty lists.
