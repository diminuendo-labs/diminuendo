**
The function `most_common_word(s)` takes a string `s` as input. It first checks if the string is empty and returns an empty string if true. Then, it initializes an empty dictionary `word_counts` to store the count of each word. The function iterates over each word in the input string (after converting it to lowercase and splitting by whitespace), updating the count in the dictionary. After counting, it finds the maximum count using `max(word_counts.values())`. It then creates a list `common_words` containing all words that have this maximum count. Finally, it returns the lexicographically smallest word from `common_words` using `min(common_words)`, which automatically handles ties alphabetically as required.
