I used a stack data structure to keep track of the opening parentheses encountered so far. When I encounter a closing parenthesis, I check if there are any opening parentheses on the stack; if there aren't, it means the closing parenthesis appears before its partner and the function immediately returns False. If the stack is empty at the end of the string, it means every opening parenthesis has been matched with a closing one, so the function returns True. This approach handles the edge case where an empty string is considered balanced by checking if the stack is empty after iterating over the entire string.
