In the function, we first check if there are less than two distinct values in the list and return `None`. If there are at least two distinct values, we sort the list in ascending order. After that, we return the second last element of the sorted list, but with an exception when the last and second last elements are equal (meaning the largest and second largest have the same value). In this case, we assume the largest is considered as the second largest to handle the edge case where the largest and second largest values are the same.
