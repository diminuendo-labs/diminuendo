I approached this task by using a simple iterative approach. I first checked for the edge case where the input list is empty, raising a ValueError in such cases. Then, I initialized variables to track the maximum value seen so far and its index. In each iteration of the loop, if the current number exceeds the max_val, I update both max_val and last_occurrence_index with their new values. This ensures that when there are duplicate largest numbers, the function returns the index of the last occurrence.
