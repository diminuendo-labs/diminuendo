The function `count_pairs` takes a list of numbers `nums` and a target number, and returns the count of index pairs that sum up to the target. We create an empty dictionary `counts` to store counts of each number in `nums`. Then we loop through the list of numbers, for each number, calculate its complement (difference between target and current number), check if it exists in the dictionary and if so, add the count of that complement to our result. Finally, we increment the count of the current number in the dictionary for each iteration and return the final result.

An edge case handled here is when duplicate numbers appear at different indices; these will not be counted as a single pair but rather as multiple distinct pairs (e.g., for [2, 2, 2] with target 4, there are three pairs: (0,1), (0,2), and (1,2)).
