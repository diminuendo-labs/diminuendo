**
- The function `longest_word` takes a single argument `s`, which is the input string.
- I first handle the edge case where the input string `s` is empty. If it is, the function returns an empty string immediately.
- For non-empty strings, I split the string into words using the `split()` method, which by default splits on whitespace.
- I then use the `max()` function with the key argument set to `len` to find the longest word in the list of words. Since `max()` returns the first maximum element encountered in case of ties, this naturally satisfies the requirement to return the last longest word in case of a tie.
- Finally, the function returns the longest word found.
