I created the function `weighted_mean` that takes two parameters: `values` and `weights`. The function first checks if either list is empty or if the total weight (sum of weights) is zero. If any of these conditions are met, it returns `None` to handle edge cases gracefully without causing a crash. Otherwise, it calculates the weighted mean by summing the product of each value and its corresponding weight, then dividing by the total weight. This ensures that the function is both correct and concise, adhering to the criteria provided.
