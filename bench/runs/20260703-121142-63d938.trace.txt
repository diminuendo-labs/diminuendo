The function `range_summary` first checks if the input list `nums` is empty and returns an empty string if true. It then sorts the unique elements of `nums` to ensure they are in ascending order. The function iterates through the sorted list, identifying consecutive ranges by checking if the current number plus one equals the next number. When a range is identified, it appends the start and end of the range to the result list; otherwise, it appends the single number. Finally, it joins all parts of the result list with commas to form the final string. This approach ensures that the output correctly represents ranges and individual numbers as specified.
