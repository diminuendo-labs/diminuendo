To implement the `chunk_pad` function, I first checked if the chunk size `k` is greater than zero to handle an edge case where the input might be invalid. Then, using a list comprehension, I split the list into chunks of size `k`. For each chunk, if it's shorter than `k`, I padded it with `None` by calculating the difference between `k` and the length of the current chunk and appending that many `None`s. This approach ensures that every chunk has exactly `k` elements, even for the last one if it comes up short.
