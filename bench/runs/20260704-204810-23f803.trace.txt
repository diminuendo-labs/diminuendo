In this solution, I first check if the input list is empty and return None in that case. If not, I remove one occurrence of the minimum and one occurrence of the maximum from the list. Then, I sort the list and calculate the index where the middle four elements will be located. I sum up these four elements and divide by the number of remaining elements (excluding the minimum and maximum) to find the trimmed mean.

The edge case I handled is when the input list contains two identical extremes, in which case removing one minimum or maximum does not affect the final result, as long as the sorted list has at least four unique elements after removal of the extremes.
