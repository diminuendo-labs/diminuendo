I used the `split()` function to create a list of whitespace-separated words, and then iterated over this list in reverse order. This way, if there are multiple longest words, I can easily find the last one by stopping at the first occurrence with the maximum length. The edge case where there's only one space in the string is handled implicitly by this approach, as it will return an empty string for a single-word input like 'hello world'.
