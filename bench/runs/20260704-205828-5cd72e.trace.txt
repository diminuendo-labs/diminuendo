I sorted the input list and found the minimum and maximum values. If there is only one occurrence of either, I computed the trimmed mean by excluding that value from the sum. If both min and max have more than one occurrence, an edge case occurs where the list is not trimmed at all, so I returned the average of the middle element and the total sum. In this case, I used the arithmetic mean since the list is not truly trimmed. The function returns None when given an empty list as specified in the task.
