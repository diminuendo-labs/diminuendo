I used a list comprehension to create chunks of size `k` from the input list. The slicing operation `lst[i:i+k]` extracts a chunk starting at index `i` with length `k`. If the chunk is shorter than `k`, I appended `None` values until it reaches the desired size using `[None]*(k-len(lst[i:i+k]))`. This ensures every chunk has exactly `k` elements. The outer loop iterates over the list in steps of `k` to ensure consecutive chunks are generated correctly.
