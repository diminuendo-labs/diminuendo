I began by handling the edge case where an empty list is passed, returning an empty string immediately. I sorted the input list to ensure that consecutive numbers are adjacent, which simplifies identifying ranges. I iterated through the sorted list, checking if each number is consecutive with the previous one. If not, I decided whether to add a single number or a range to the `ranges` list. After processing all numbers, I appended the last segment (which might be a single number). Finally, I joined all segments into a comma-separated string and returned it. This approach ensures that the function is both correct and concise while being easy to understand.
