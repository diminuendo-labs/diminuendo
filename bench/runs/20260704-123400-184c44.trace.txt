Firstly, I sorted the input list `nums`. Then, I calculated the median index of the list and checked if it is an even or odd number. Based on that information, I determined the indices for the second smallest and second largest numbers (excluding min and max). Afterward, I calculated the trimmed mean by summing up the numbers from the selected indices and dividing them by the difference between the maximum and minimum indices to exclude. Lastly, the function returns the result as a float. The edge case handled is when the input list is empty; in this situation, the function will return `None`.
