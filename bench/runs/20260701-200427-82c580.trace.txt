I created a function `rle(s)` that takes a string `s` as input and returns its run-length encoding. The function first checks if the input string is empty, and if so, it immediately returns an empty string to handle this edge case. If the string is not empty, it initializes an empty list `encoded` to store parts of the encoded string and a counter `count` set to 1. It then iterates through the string starting from the second character, comparing each character with the previous one. If they are the same, it increments the count; otherwise, it appends the previous character and its count to the `encoded` list and resets the count to 1. After the loop, it appends the last character and its count to handle the final group. Finally, it joins all parts of the encoded string into a single string and returns it.
