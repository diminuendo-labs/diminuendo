The function `longest_word` first checks if the input string `s` is empty. If it is, it returns an empty string. Otherwise, it splits the string into a list of words using the `split()` method. It then determines the maximum length of any word in the list using a generator expression. Finally, it iterates over the words in reverse order to find and return the last longest word. This approach ensures that ties are broken by returning the last occurrence of the longest word as specified in the task criteria.
