First, I defined the set of delimiters as a comma-separated string and split it into a list. Then, using the `itertools.groupby()` function, I grouped consecutive characters based on whether they are in the delimiter list. Any group with an empty first element (meaning no character belongs to the delimiter set) was filtered out. The remaining groups were joined together and stored as a list of chunks.

The edge case I handled is when there's only one empty string in the final result, which would cause the function to return an empty string instead of a single comma. To address this issue, I added a conditional statement that checks if the last chunk (`chunks[-1]`) is an empty string and, if so, appends a single empty string ("") before joining all pieces together with commas. This ensures the final result will always contain at least one comma.
