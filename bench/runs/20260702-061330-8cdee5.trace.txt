**  
I started by checking if the input list `nums` is empty and returning an empty string if it is, as that's our edge case. Then, I sorted the list to ensure it's in ascending order and removed duplicates using `set`. I used a while loop to iterate through the sorted numbers, identifying ranges by checking if the next number is exactly one greater than the current one. If a range was identified, it was formatted as 'start-end' and added to the result list; otherwise, a single number was added. Finally, the parts of the result were joined with commas and returned as the final string.
