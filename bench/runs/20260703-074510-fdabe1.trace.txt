I created a function `depth_max` that iterates through each character in the input string `s`. It maintains two variables, `max_depth` to keep track of the maximum nesting depth encountered and `current_depth` for the current level of nesting. For every opening bracket `[`, it increments `current_depth` and updates `max_depth` if necessary. For every closing bracket `]`, it decrements `current_depth`. If `current_depth` becomes negative (indicating a closing bracket appears before its opener), or is non-zero at the end (indicating an unclosed opener), the function returns `-1`. This ensures that only valid sequences of brackets are processed correctly. The edge case of an empty string, which should return `0`, is handled explicitly with an assertion.
