I defined the function `median` that takes a list of numbers as input. First, I checked if the list is empty and returned `None` immediately to handle the edge case where there are no elements. Then, I sorted the list to arrange the numbers in ascending order. Depending on whether the length of the list is odd or even, I calculated the median accordingly: for an even number of elements, I returned the average of the two middle numbers; for an odd number, I returned the middle number as a float. This solution ensures correctness and readability while handling the edge case where the input list might be empty.
