The function `longest_word(s)` first checks if the input string `s` is empty. If it is, it returns an empty string. The input string is then split into a list of words using the `split()` method, which by default splits on whitespace. The longest word is determined using Python's built-in `max()` function with the key parameter set to `len`, ensuring that the longest word based on length is found. In case there are ties (multiple words of the same maximum length), a list comprehension is used to filter out all words that match this length, but in reverse order, so that the last such word is selected by default when slicing with `[0]`. This approach ensures the function is concise and readable while correctly handling edge cases.
