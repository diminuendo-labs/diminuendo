I started by checking if the input list `nums` is empty and returning `None` if it is, as this is an edge case that needs to be handled separately. If the list is not empty, I sorted the list using Python's built-in `sorted()` function. I then determined the middle index of the list. If the length of the list is even, I returned the average of the two middle numbers as a float by summing them and dividing by 2.0. If the length of the list is odd, I simply returned the value at the middle index. This approach ensures that the function works correctly for both even and odd lengths of input lists and handles the edge case of an empty list.
