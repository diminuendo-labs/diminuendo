I approached this problem by first splitting the input string into a list of words using the `split()` method. Then, I iterated over each word and checked if it was entirely uppercase using the `isupper()` method. If it was, I added it to the result list as is; otherwise, I converted it to title case using the `title()` method and added it to the list. Finally, I joined the processed words back into a string with single spaces between them using the `join()` method. This approach allowed me to handle edge cases like all-uppercase words while maintaining concise and readable code.
