I used a stack data structure to handle the parentheses in the given string. Whenever an opening bracket ('(') is encountered, it gets pushed onto the stack. For closing brackets ('）'), if the stack is empty or the top element of the stack isn't an opening bracket, then the function returns False (indicating the parentheses are not balanced). If the loop finishes without returning early and the stack is still empty, it indicates that all parentheses are balanced and the function returns True. The edge case I handled is when there are unmatched or misplaced parentheses in the string.
