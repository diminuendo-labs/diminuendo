To tackle the task, I utilized Python's `re.split` method with a regular expression that matches either commas or semicolons. This allows us to handle any run of consecutive delimiters as a single split point. Following this, I filtered out any empty strings from the resulting list and joined the remaining pieces with a single comma using `','.join()`. This approach ensures the function is concise and readable while correctly handling various edge cases, including multiple consecutive delimiters at the beginning or end of the input string.
