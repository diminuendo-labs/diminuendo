I chose to use a set to remove duplicates and then sort the list. The function iterates through the sorted list, identifying consecutive ranges by checking if the next number is exactly one greater than the current. When a range is identified, it's formatted as 'start-end' or just 'start' if they are the same, and added to the result list. Finally, all elements in the result list are joined into a single string with commas separating them. This approach ensures that the function works efficiently even for larger lists and handles edge cases like an empty input gracefully.
