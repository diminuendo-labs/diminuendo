The function `chunk_pad` splits the input list into chunks of size `k`. It uses list slicing (`lst[i:i+k]`) to get each chunk and then appends `None` values to pad the last chunk if it is shorter than `k` elements. This approach ensures that every chunk has exactly `k` elements. The edge case where the input list is empty is handled by returning an empty list, which meets the requirement that an empty list should return [].
