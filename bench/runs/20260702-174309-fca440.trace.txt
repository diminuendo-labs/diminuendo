I approached this task by first breaking down the problem into smaller sub-tasks: splitting the input string into words, counting the occurrences of each word, and finding the maximum count. To handle the edge case where the largest value appears more than once, I used Python's built-in `Counter` class to keep track of the counts, then filtered out words with the maximum count from the list in reverse order. This ensures that if multiple words have the same maximum count, their last occurrence is returned.
