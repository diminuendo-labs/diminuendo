I first checked if the input list `words` is empty and returned `None` as an edge case. For non-empty lists, I used Python's `sorted()` function with a custom key to sort the words. The key sorts primarily by negative length (to get increasing order of length) and secondarily by reverse alphabetization (`x[::-1]`). Finally, I returned the k-th word if `k` is within the valid range, otherwise, I returned `None`.
