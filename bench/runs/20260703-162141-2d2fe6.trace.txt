The function takes a list `nums` as input and returns the sum of its distinct values. First, we convert the list into a set to eliminate duplicates since sets in Python do not allow duplicate elements. Then, we use the built-in `sum()` function to calculate the total sum of the distinct numbers in the set. In case the list is empty, the function returns 0 as per the given criteria. The named edge case handled here is an empty list input.
