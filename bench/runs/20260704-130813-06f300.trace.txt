I used a simple loop to split the underscore-delimited identifier into individual tokens. Then I created an exceptions list containing the strings 'URL' and 'ID'. For each token, I check if it is in the exception list or the first token (i.e., index 0). If not, I capitalize it using `capitalize()`. If the token is a valid exception, I leave it unchanged.

To handle edge cases, I checked for the identifier being entirely uppercase and return it in lowercase if that's the case. This ensures that URL and ID remain in their original form even when they are the first token in the identifier.
