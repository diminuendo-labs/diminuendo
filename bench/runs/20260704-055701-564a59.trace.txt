I chose to use a stack data structure to keep track of the opening parentheses. As I iterate through the string, whenever I encounter an opening parenthesis, I push it onto the stack. When I encounter a closing parenthesis, I check if the stack is empty; if it is, that means there's no matching opening parenthesis, so I return False immediately. If the stack is not empty, I pop the top element off the stack (which should be the corresponding opening parenthesis). Finally, after iterating through the entire string, I check if the stack is still empty; if it is, then all parentheses were properly balanced, and I return True.
