I approached this task by first removing duplicate values in the list using the set() function and converting it back to a list for sorting. I then implemented a simple conditional statement that checks if n is within the valid range (greater than 0 and less than or equal to the number of unique elements) before returning the nth smallest element. An edge case to handle is when n is less than 1 or exceeds the number of distinct values in the list, in which case I return None.
