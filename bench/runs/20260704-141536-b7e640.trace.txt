I used the `re.split` function from Python's `re` module to split the string based on either commas or semicolons. This approach handles multiple consecutive delimiters by treating them as one split, which simplifies the logic needed to drop empty pieces and join them with a single comma. The use of a regular expression makes it easy to specify that both commas and semicolons are considered delimiters. Additionally, I used a generator expression inside `join` to filter out any empty strings that might be created by leading or trailing delimiters in the input string, ensuring that the output does not start or end with an unwanted comma.
