The function `longest_word(s)` takes a string `s` as input and returns the longest whitespace-separated word. If there are ties, it returns the last such word. An empty string or a string with only whitespace returns an empty string.

1. **Edge Case Handling:** The function first checks if the input string is empty or contains only whitespace using `strip()`. If so, it immediately returns an empty string.
2. **Splitting the String:** The input string is split into a list of words using `split()`, which handles multiple spaces and leading/trailing whitespace by default.
3. **Finding the Longest Word:** It uses the `max()` function with the key argument set to `len` to find the longest word in the list.
4. **Handling Ties:** To handle ties, it creates a list of all words that have the maximum length found earlier. This is done by iterating over the list in reverse (to get the last such word on tie) and checking their lengths against the maximum length.
5. The function returns the first element from this list, which will be the last longest word encountered on ties.

This approach ensures correctness, conciseness, and readability while handling the specified edge case.
