To implement the `snake_to_camel` function, I used Python's string manipulation capabilities. The function splits the input string `s` by underscores (`_`) into a list of words, filters out any empty strings (which can occur due to leading, trailing, or repeated underscores), and then capitalizes each word. Finally, it joins these capitalized words together without any separators to form the camelCase string. This approach ensures that all edge cases, including leading, trailing, and repeated underscores, are handled correctly, as well as an empty input string returning an empty output.
