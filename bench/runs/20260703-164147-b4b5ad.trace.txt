I first split the input string by whitespace into a list of words. Then, I iterate through each word and check if it is entirely uppercase using `isupper()`. If it's not entirely uppercase, I title-case it (first letter upper, the rest lower). Finally, I collapse all runs of multiple spaces into single spaces by joining the list back into a string with a single space separator. The edge case handled here is that words already entirely uppercase stay exactly as they are.
