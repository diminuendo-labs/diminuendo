I created the function `balanced(s)` that iterates through each character in the input string `s`. For every opening parenthesis `'('`, I increment the `balance` counter. For every closing parenthesis `')'`, I decrement it. If at any point the balance goes negative, it means a closing parenthesis appears before its matching opening one, so the function returns `False` immediately. After iterating through all characters, if the final balance is zero, the parentheses are perfectly balanced and the function returns `True`. This approach ensures the code is concise and readable while handling the edge case of an unbalanced string with too many closing parentheses before any opening ones.
