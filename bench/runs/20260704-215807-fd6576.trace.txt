The function first checks if the input list is empty and returns `None` in such a case. Then it sorts the list and calculates the indexes of minimum and maximum values, which are the second and penultimate elements, respectively. If the list has less than 3 elements, it simply returns the mean of all elements. Otherwise, it computes the sum of the elements from the second to the penultimate position in the sorted list, divides this sum by the number of elements being considered (excluding the minimum and maximum), and returns the trimmed mean as a float.

One edge case that is handled is when the input list contains less than 3 elements, where it returns the simple average instead of raising an exception or returning `None`.
