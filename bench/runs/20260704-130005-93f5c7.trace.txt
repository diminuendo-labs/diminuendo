The function first splits the input string by underscores to obtain tokens. It creates an empty list `result` to store transformed tokens. Then, it iterates through each token and checks if it's in the exception list or an empty string. If it doesn't belong to the exception list, it converts the first character of the token to uppercase if there are existing elements in the result list, or appends it to the result list with lowercase for the initial token.
The function also handles an edge case where 'URL' or 'ID' tokens appear anywhere within the input string by adding a check for them during each iteration and replacing the current element in the `result` list if found. Finally, the function returns the joined result without any separators.
