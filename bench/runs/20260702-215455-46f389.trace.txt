The function `max_index` first checks for the edge case where the input list is empty. If it is, the function returns `None`. For a non-empty list, it finds the maximum value using Python's built-in `max()` function. It then finds the index of this maximum value from the end of the list by reversing the list and using the `index()` method. The last occurrence index is adjusted to be from the original list's perspective by subtracting the reversed index from the total length minus one.
