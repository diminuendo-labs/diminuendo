The function `depth_max(s)` takes a string `s` as input and calculates the maximum nesting depth of square brackets. It uses two variables to keep track of the current depth and the maximum depth found so far. The inner loop iterates through each character in the string, incrementing or decrementing the depth accordingly when encountering an opening or closing bracket, respectively. An edge case is handled by checking if a closing bracket appears before its opener, in which case the function returns -1. To ensure that the function can handle an empty string correctly, it initializes the maximum depth to 0 instead of None.
