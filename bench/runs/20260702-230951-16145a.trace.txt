I used the `Counter` class from the `collections` module to count the occurrences of each word in the input string. This allows for efficient counting and comparison of word frequencies. To break ties alphabetically, I used a lambda function as the key argument in the `max` function, which returns both the frequency and the word itself. If the input string is empty, an edge case handled by this function is that it immediately returns an empty string without attempting to process the input.
