In this function, I first sorted the list in descending order to arrange all elements from largest to smallest. If there are less than two distinct values (i.e., the list is either empty or contains only one element), I return None as there isn't a second-largest value. Otherwise, I return the second element of the sorted list, which represents the second-largest distinct value in the original input list. To handle the edge case where the list is either empty or contains only one unique element, I included that condition when checking for less than two distinct values before returning None.
