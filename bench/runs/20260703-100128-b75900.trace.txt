** I implemented a function `longest_word(s)` that splits the input string `s` into words. It iterates through each word, keeping track of the maximum length encountered and collecting all words with that maximum length. If there are ties in length, it appends the current word to the list of longest words. Finally, it returns the last word from this list, which is the last longest word due to Python's list indexing starting at 0. The edge case where `s` is an empty string is handled by returning an empty string right away.
