The function `range_summary` first handles the edge case of an empty list by returning an empty string. It then sorts and deduplicates the input list to ensure we are working with unique, sorted values. The main logic iterates through this list, identifying consecutive sequences using a while loop. When it finds a sequence (either single or multiple numbers), it appends the appropriate format ('start-end' for ranges, 'number' otherwise) to the `ranges` list. Finally, it joins all elements of `ranges` with commas to form and return the final string. This approach ensures correctness by handling all edge cases and is concise yet readable.
