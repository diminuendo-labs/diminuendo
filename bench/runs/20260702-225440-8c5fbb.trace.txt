I wrote the `median` function by first checking if the input list is empty and returning None in that case. Then I used Python's built-in `sorted` function to sort the list, which has a time complexity of O(n log n). Next, I checked if the length of the sorted list is odd or even and returned either the middle value or the average of the two middle values as a float. This approach is concise and readable, and handles the edge case where the input list is empty.
