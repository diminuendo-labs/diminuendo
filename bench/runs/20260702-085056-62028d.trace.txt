I implemented the `median` function by first checking if the input list `nums` is empty, returning `None` in that case to handle an edge case explicitly. If the list is not empty, I sorted it and then determined the middle index. Depending on whether the length of the list is odd or even, I returned either the middle element directly or the average of the two middle elements as a float. This approach ensures the function correctly calculates the median for both odd and even lengths of input lists while handling an empty list appropriately.
