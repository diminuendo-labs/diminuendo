I used Python's built-in `split()` function to separate the input string into a list of words. Then I created a conditional statement to check if the number of requested words (n) is greater than zero, and if so, returned the first n words joined by single spaces using the `join()` function. If n is equal to or less than zero, I returned an empty string ''. To handle an edge case, when the input string has fewer words than requested, the function returns all the words available from the given string.
