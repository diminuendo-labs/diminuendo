The approach used here is a combination of list comprehension and the built-in `isinstance()` function. The double-for loop first iterates over each sublist within the input list (lst). If a sublist is found, its contents are yielded one by one, otherwise, the element remains unchanged. This ensures that only one level of nesting gets flattened and any deeper nesting stays intact. One edge case handled by this function is when there are no nested lists in the input list; in such a case, the original list will be returned as-is.
