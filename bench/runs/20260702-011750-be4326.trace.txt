I defined a function `median` that takes a list of numbers as input and returns the median value. The function first checks if the input list is empty, returning `None` in this case to handle an edge condition. If the list has at least one element, it sorts the list and then uses conditional logic to determine whether the length of the list is odd or even. When the length is odd, it returns the middle value as a float; when the length is even, it returns the average of the two middle values as a float. I chose to use floating-point arithmetic to ensure that the result can be represented exactly for all possible input lists.
