The function `max_index` first checks if the input list `nums` is empty. If it is, it returns `-1` as an edge case handling. For non-empty lists, it finds the maximum value using Python's built-in `max()` function. To handle cases where the maximum value appears multiple times and we need the index of its last occurrence, I reverse the list (`nums[::-1]`) and use the `index()` method to find the first occurrence of the maximum value in this reversed list. Since reversing shifts the indices, I then calculate the correct original index by subtracting the found index from the total length of the original list minus one.
