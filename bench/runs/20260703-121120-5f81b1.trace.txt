The function `longest_word(s)` processes the input string to find and return the longest whitespace-separated word. If the input string is empty, it returns an empty string. The function splits the input string into words using `split()`, which handles multiple consecutive spaces correctly by returning only non-empty strings. It then iterates through the words in reverse order, updating the current longest word when a longer one (or a tie) is found. By iterating in reverse, we naturally prioritize the last occurrence of the longest word in case of ties. This approach ensures that the function adheres to the requirement to return the last such word.
