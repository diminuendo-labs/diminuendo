I implemented the `average` function to calculate the arithmetic mean of a list of numbers. The function first checks if the input list `nums` is empty using `if not nums`. If it is, the function returns `None` as specified. Otherwise, it calculates the sum of all numbers in the list using `sum(nums)` and divides by the length of the list using `len(nums)`, returning the result as a float. This approach ensures that the function handles both non-empty and empty lists correctly, with the edge case of an empty list explicitly addressed.
