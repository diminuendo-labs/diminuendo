**
The function `range_summary` takes a list of integers, sorts it, and then iterates through the sorted list to identify consecutive numbers. It uses two pointers (`start` and `end`) to track the current range being processed. When a non-consecutive number is found, the current range (or single number) is added to the result list in the appropriate format ('start-end' or 'single_number'). After processing all numbers, any remaining range is added to the list. Finally, the ranges are joined into a single string separated by commas and returned. This approach ensures that the function handles edge cases such as empty lists and consecutive runs efficiently.
