I created a function named `clamp` that takes three arguments: the value to clamp, and the lower and upper bounds. I first check if the lower bound is greater than the upper bound, in which case the range is degenerate, so I return `None`. Otherwise, I compare the input value with the bounds and return the corresponding bound if the input value is outside the range, otherwise I simply return the input value itself. The edge case handled here is when the input value is less than the lower bound or greater than the upper bound.
