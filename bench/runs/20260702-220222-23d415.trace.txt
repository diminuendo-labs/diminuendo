I used a list comprehension to iterate over each word in the input string `s` after splitting it by whitespace. For each word, I checked if it was already entirely uppercase. If so, I kept it as is. Otherwise, I title-cased it (making the first letter uppercase and the rest lowercase). Finally, I joined these processed words back into a single string with spaces in between. The edge case of handling a single uppercase word is demonstrated with the print statement at the end.
