The approach is to first create a set of unique numbers in the given list to remove duplicates and then sort it. After that, we check if the number of elements in the sorted list is less than 1 or greater than the requested nth smallest number. If so, return None. Otherwise, return the nth smallest distinct value by indexing the sorted list with n - 1. The edge case considered here is when the input list has no unique numbers (i.e., all numbers are duplicates), in which case we return None as well.
