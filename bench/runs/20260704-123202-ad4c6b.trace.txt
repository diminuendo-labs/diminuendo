The approach for this function is to first sort the input list `nums`. If the list is empty, we return `None`, as requested. After sorting, we find the median index by dividing the length of the list by 2 and rounding down, ensuring that in case of an odd number of elements, the middle element will be included. We then calculate the sum of all numbers except for the minimum and maximum values, which are the first and last elements of the sorted list when the median index is not at the beginning or end. Finally, we divide this sum by 2 to find the mean of these numbers, giving us the trimmed mean. An edge case handled here is that of an empty list.
