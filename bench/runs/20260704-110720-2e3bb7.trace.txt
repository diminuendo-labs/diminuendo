I implemented the function using a stack data structure to keep track of the opening parentheses encountered. As I iterate through the string, when an opening parenthesis is found, it's pushed onto the stack, and when a closing parenthesis is found, the top element (the corresponding opening parenthesis) is popped from the stack. If at any point there are no more elements on the stack to match with an extra closing parenthesis, the function returns False immediately. The function then returns True if the stack is empty after checking the entire string, indicating balanced parentheses.
