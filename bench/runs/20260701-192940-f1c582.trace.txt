The approach involves creating a dictionary to count the occurrences of each character in the string. This ensures that we can efficiently determine which characters are unique (i.e., have only one occurrence). After counting, we iterate through the string again to find the first character with a count of 1 and return its index. If no such character exists, the function returns -1. The edge case where the input string is empty is handled by returning -1 immediately if the string is empty.
