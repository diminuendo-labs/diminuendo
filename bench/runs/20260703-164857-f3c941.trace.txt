I created a set to remove duplicates and count the frequency of each word using a dictionary comprehension. Then I found the minimum value (frequency) and corresponding key (least frequent word) in the dictionary using the built-in `min()` function, breaking ties by sorting the keys alphabetically. An empty string is returned when no words are present in the input string. One edge case handled is when all words have the same frequency; in this situation, the least frequent word will be chosen alphabetically first.
