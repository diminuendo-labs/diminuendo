The function `weighted_mean` first checks if either the `values` list is empty, the `weights` list is empty, or the sum of the weights is zero. If any of these conditions are true, it returns `None`. Otherwise, it calculates the weighted mean using a generator expression that multiplies corresponding values and weights together and then sums up these products. The result is then divided by the total weight to get the final weighted arithmetic mean. This function avoids division by zero and ensures graceful handling of edge cases where the input lists might be empty or have a total weight of zero.
