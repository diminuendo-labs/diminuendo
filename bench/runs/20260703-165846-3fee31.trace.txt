I decided to create a function `clamp(value, lo, hi)` that first checks if the lower bound (lo) is greater than the higher bound (hi), and in this case, returns None. If the bounds are valid, it uses Python's built-in max() and min() functions to find the minimum of the three values (value, lo, hi) and the maximum between value and hi. This ensures that the clamped value remains within the specified range. Additionally, I handled an edge case where the value is outside the range, in which case it raises a ValueError with a custom message.
