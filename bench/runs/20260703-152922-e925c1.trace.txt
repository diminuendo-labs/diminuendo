To implement the `range_summary` function, I first sorted the unique elements of the input list to ensure they are in ascending order. Then, I used a while loop to iterate through the sorted numbers and identified consecutive sequences by checking if the next number is exactly one more than the current number. For each sequence, I determined whether it was a single number or a range and appended the appropriate string to the `ranges` list. Finally, I joined all the strings in the list with commas to form the final output. This approach ensures that the function handles consecutive numbers correctly and produces the desired formatted string. The edge case of an empty input list is handled by returning an empty string immediately.
