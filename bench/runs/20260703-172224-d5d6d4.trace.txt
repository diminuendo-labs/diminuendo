The function `longest_word(s)` first splits the input string `s` into a list of words using the space as separator. Then it finds the maximum length of any word in the list and returns the word with that maximum length, or an empty string if there are no words (edge case: empty string). If multiple words have the same maximum length, it returns the last one among them. The code is concise and readable by following Python's built-in functions like `split()`, `max()` and lambda expressions.
