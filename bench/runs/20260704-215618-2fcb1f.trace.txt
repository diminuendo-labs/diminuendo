The function first checks if the input list is empty and returns `None` in that case. Then it sorts the list, calculates the median index, and handles the edge case where the list length is even by defining two indices for the median values. It replaces the minimum and maximum values in the sorted list with `None`, removes the `None` elements, re-sorts the list, and finally computes the mean of the trimmed list. The result is returned as a float. This approach ensures that at least one occurrence of the minimum and maximum values are removed before calculating the mean.
