The function, `split_csvish`, takes a string `s` as input and returns a list of fields obtained by splitting the string on commas (`','`). The whitespace around each field is trimmed using the `strip()` method. Empty fields after trimming are dropped using the conditional statement `if field`. An empty list `['']` is returned if all fields are emptied after trimming. This function handles an edge case where there may not be any comma-separated fields in the input string.
