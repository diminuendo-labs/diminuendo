I first checked if the input list is empty and returned an empty list immediately to handle this edge case. Then, I used a list comprehension to create chunks of size `k` by slicing the original list. For the final chunk, which may be shorter than `k`, I calculated how many `None`s needed to pad it using modulo operation and created a new list with these `None`s to append to the result. This approach ensures that every chunk is exactly `k` elements long, even if the last one requires padding.
