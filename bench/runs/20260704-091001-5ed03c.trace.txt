The function `median(nums)` first checks if the input list is empty, and returns `None` if it is. It then sorts the list and calculates the middle index (`mid`) by dividing the length of the list by 2. If the number of elements in the list is odd, the function returns the middle element. If the number of elements is even, the function calculates and returns the average of the two middle elements. An edge case handled here is when the input list has zero elements.
