The function `kth_ordered` takes a list of words and an integer `k`. It first checks if `k` is within the valid range (1 to the length of the list). If not, it returns `None`.

To sort the words according to the given criteria:
- We use Python's built-in `sorted()` function.
- The key for sorting is a tuple where the first element is negative length (`-len(x)`) to order by increasing length in ascending order.
- The second element is the reverse of each word (`x[::-1]`) to ensure that words of equal length are ordered reverse-alphabetically.

Finally, it returns the k-th word from the sorted list (adjusting for zero-based indexing with `k - 1`).
