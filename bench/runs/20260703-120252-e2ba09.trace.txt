The function `range_summary` first checks if the input list is empty and returns an empty string immediately if it is. It then sorts the unique numbers from the list to ensure they are in ascending order. The main logic iterates through these sorted numbers, identifying consecutive sequences by checking if the next number is exactly one more than the current one. When a sequence is identified, either a single number or a range (start-end) is added to the result list. Finally, all parts of the result list are joined into a comma-separated string and returned. The edge case where the input list might be empty is explicitly handled at the beginning of the function.
