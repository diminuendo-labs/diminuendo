I used a Stack Data Structure to solve this problem. Whenever an opening parenthesis ('(') is encountered, it gets pushed onto the stack. When a closing parenthesis ('))' is found, if the stack is empty or the top element of the stack isn't an opening parenthesis, the function returns False, indicating that the parentheses are not balanced. If no such situation arises, meaning all parentheses are properly nested and the stack is empty at the end, the function returns True. I chose this approach because it allows for easy handling of any number of nested parentheses and efficiently checks for balance in a single pass through the input string. The edge case handled here is when there is an unbalanced pair of parentheses like ')' or '()))'.
