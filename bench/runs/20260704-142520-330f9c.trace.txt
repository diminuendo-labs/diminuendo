**

The function `kth_ordered` first checks if `k` is within the valid range. If not, it returns `None`. It then sorts the list of words using a custom key. The key is a tuple where the first element is the length of the word (to sort by increasing length), and the second element is the negative ASCII value of the first character of the word (to sort reverse-alphabetically among words of equal length). Finally, it returns the `k-th` word from the sorted list.
