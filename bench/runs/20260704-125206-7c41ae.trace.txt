I first defined the delimiters as a string containing all possible separators. Then, I split the input string on those delimiters and removed any empty strings at the start and end of the resulting list. To handle the edge case where a string starts with a delimiter, I created a new list and concatenated consecutive pieces that didn't start with a delimiter. Finally, I rejoined the pieces with a single comma between each. This approach ensures a concise and readable solution while handling the specified edge case.
