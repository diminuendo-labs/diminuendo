The function `longest_word(s)` splits the input string `s` into a list of words using the whitespace as delimiter. It then finds the maximum length of any word in the list using the `max()` function with the `len()` function applied to each word and the built-in `map()` function. The longest word is found by filtering the list of words based on their lengths (using a lambda function) and choosing the one with the maximum length, or if there's a tie, selecting the last such word using the `max()` function with a custom key comparison function. To handle the edge case where the input string is empty, it returns an empty string ''.
