I chose to write a simple iterative function that uses a stack data structure. As I iterate through the string, whenever I encounter an opening parenthesis, I push it onto the stack. Whenever I encounter a closing parenthesis, I check if the stack is empty (which would be an edge case where there are more closing parentheses than opening ones) and pop the top element from the stack. Finally, after iterating through the entire string, I return whether the stack is empty or not – if it's empty, that means every opening parenthesis had a corresponding closing one, so the function returns True, indicating balanced parentheses; otherwise, it returns False.
