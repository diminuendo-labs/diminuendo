The function iterates through the input list and keeps track of runs by checking if each number is one more than the previous one. If it's not, a new run starts. We store the start and end of each run in a list as tuples for consecutive runs. Then we join these runs with commas or hyphens depending on whether they are single numbers or ranges. For edge cases like [1, 2, 2, 3], it correctly identifies two runs: [1, 2] and [3].
