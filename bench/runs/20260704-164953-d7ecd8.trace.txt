**

The `chunk_pad` function takes a list `lst` and an integer `k`. It first checks if the list is empty and returns an empty list if true. It then iterates through the list in steps of size `k`, creating chunks of up to `k` elements each. If a chunk is smaller than `k`, it pads it with `None` until it reaches the required length. These padded chunks are collected into the result list, which is returned at the end. This function handles the edge case where an empty list is passed by immediately returning an empty list without any processing.
