I created an empty list named `runs` to store the summarized ranges of consecutive integers found in the input list. I set a starting and ending number to the first element of the input list, and iterate through the list checking if the current number is different from the end of the previous range. If it is, I append the summarized range to the `runs` list, reset the start and end variables for the next run, and continue this process until all numbers in the list have been processed. Finally, I join the ranges in the `runs` list with commas and return the string.

An edge case that is handled by this function is when the input list is empty; in such a case, it returns an empty string instead of raising an error.
