I approached this task by first handling the edge case where an empty string is input, returning an empty string immediately. Then, I converted the entire string to lowercase and split it into individual words. Using a dictionary, I counted the occurrences of each word. Finally, I sorted the items in the dictionary based on their counts (in descending order) and then alphabetically. The most common word, which is the first item in this sorted list, is returned. This solution ensures that ties are broken alphabetically as required.
