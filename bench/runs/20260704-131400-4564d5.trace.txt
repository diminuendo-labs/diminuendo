First, I sorted the list of words by increasing length using a built-in Python function `sorted()`. To handle the edge case where multiple words have the same length, I applied a custom key function to sort them reverse-alphabetically within their respective length groups. After that, I calculated the total length of all words for determining whether the desired index is in range. Lastly, I found the segment where the desired index falls using a loop and sliced out the corresponding word from the sorted list.
