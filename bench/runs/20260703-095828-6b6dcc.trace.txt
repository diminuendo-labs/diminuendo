I defined the function `longest_word(s)` which takes a string `s` as input. The function first checks if the string is empty and returns an empty string if true, handling this edge case immediately. It then splits the string into words using `split()`. To find the longest word, it calculates the maximum length of the words. Next, it filters out all words that are not of this maximum length and since we want the last word in case of a tie, it reverses the list before filtering. Finally, it returns the first element of the filtered list if it exists; otherwise, it returns an empty string. This solution is concise and leverages Python's built-in functions for clarity and efficiency.
