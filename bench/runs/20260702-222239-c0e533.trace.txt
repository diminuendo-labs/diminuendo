To implement the `range_summary` function, I first handle the edge case where the input list is empty, returning an empty string immediately. Then, I sort and deduplicate the input list to ensure all numbers are unique and in ascending order. I initialize `start` and `end` pointers to the first number of the sorted list. As I iterate through the remaining numbers, I check if the current number forms a consecutive range with the previous one. If not, I append the current range (or single number) to the result list and reset the `start` and `end` pointers. Finally, after exiting the loop, I add the last range to the result list and return it as a comma-separated string. This approach ensures that consecutive numbers are collapsed into ranges while single numbers remain unchanged, adhering to the specified criteria.
