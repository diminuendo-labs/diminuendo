The function first strips any leading or trailing whitespace from the input string. It then splits the string using a combination of commas, semicolons, and spaces as delimiters. To handle the edge case where consecutive delimiters are mixed, we check if the next piece should be combined with the current one based on their last and first characters, respectively. Finally, empty pieces at the start and end are dropped, and the remaining pieces are rejoined with a comma between each.
