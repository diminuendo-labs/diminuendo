To solve this task, I first checked if the given list `nums` is empty. If it is, then the function returns `None`. Otherwise, it calculates the arithmetic mean by taking the sum of all elements in the list and dividing it by the number of elements (i.e., the length of the list).
To ensure that the function also handles the edge case where the mean is a whole number but not zero (i.e., all numbers in the list are identical), I use the `round()` function to round the result to six decimal places when the calculated mean equals its integer value. This guarantees that the function will return a non-zero float even for this special case.
